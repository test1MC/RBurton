Creating Backordered product reports

Use this report to identify order items that have not been filled because they have been placed in a back-order status. This report shows:

  • Order Status
  • Order Number (click on the order number to pull up the order details page in a modal window)
  • Product Code
  • In Stock Date (if known)
  • UPC
  • Part Number
  • Alternate Part Number (if applicable)

To create a Backordered product report

  1. In Analytics, select Product Reports > Back Ordered Products.
