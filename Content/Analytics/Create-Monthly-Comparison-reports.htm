Creating Monthly Comparison reports

Create Monthly Comparison reports to see sales trends by month. You can select one or more years to compare. From within the chart, you can display specific years by clicking on the corresponding year icon at the bottom of the chart.

To create a Monthly Comparison report

  1. In Analytics, select Order Reports > Monthly Comparison.Use the menus to filter order data.
  2. In the Years menu, select one or more years to compare data. Each year will be designated by a different color.

  3. Select the Report By menu to display different order data. You can choose from Total Orders by quantity, Total Orders by dollar amount, Average Order Value, Total Items Sold, or Credited orders by dollar amount.

  4. In Order Status you can filter data by displaying All Orders, Active or Complete Orders, or Complete Orders only.

  5. From Order Type choose the sites to display data from.

  6. Select Tier Groups if you have customer tier groups set up.
  7. To compare order data by salesperson, select the check box for each salesperson you want to include in the comparison.

  8. Use Filter By to filter according to additional sets of information such as Brands, Affiliates, or Sub-Categories.
