Creating Shipping Latency reports

This report displays a pie graph broken down by number of days shipping for the selected time period. Use this report to see at how quickly order items are moving through order processing to actual shipment.

A pie chart shows shipping latency for orders items within the selected date range that have not yet left the warehouse.

To create a Shipping Latency report

  1. In Analytics, select Order Reports > Shipping Latency.
  3. If Tier Groups are set up on your site, you can select to filter your search by specific group, all groups, all customers, and so on.
  4. Click the Search button.
  5. Click on any chart segment to display a table showing all items that shipped in that period of time, along with additional information related to each item.
