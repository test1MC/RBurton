Creating Hourly Trends reports

Use the Hourly Trends report to view the number of orders processed by your site each hour per day. The numbers on the left side of the chart show the number of orders. Administrators can exclude the actions of users from specific IP addresses in the report results. Manage IP addresses from Users > Website Settings > General > General Information.

The numbers on the right side of the chart show the total dollar amount of the orders. The hourly breakdown is displayed in the lower portion of the page. You can see the total number of orders per hour, and the total dollar value for those orders.

To create an Hourly Trends report

  1. In Analytics, select Order Reports > Hourly Trends.
  2. Select the Date Range and the Order Type from the lists.
  3. Click Search to filter the results.
  4. Click on the Orders or Dollars icons at the bottom of the chart to display either set of charts, or both charts.
