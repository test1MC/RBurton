Creating Top Products reports

Use the Top Products report to display a list of your most popular items. It also gives a breakdown of order totals and shipping for those items. You can generate six types of top product reports based on the Report Options you select:

Products by revenue

This report displays the top selling products for the filtered time period. It displays a bar chart with the merchandise total for each item. Detailed information appears below the chart, including SKU, views, orders, merchandise totals, tax, shipping, and order totals. Click View Details (binoculars) to view option details, if applicable, for the product. This report can be exported to an Excel spreadsheet.

 Products by views

This report displays the sales for the viewed products for the filtered time period. It displays a bar chart with the merchandise total for each item. Detailed information appears below the chart, including SKU, views, orders, merchandise totals, tax, shipping, and order totals. Click View Details (binoculars) to view option details, if applicable for the product. This report can be exported to an Excel spreadsheet.

 Specify a product

This report displays the sales for the selected product for the filtered time period. A bar chart a the top of the page displays the merchandise total for the item. Detailed information appears below the bar chart, including: SKU, views, orders, merchandise totals, tax, shipping, and order totals. Click View Details (binoculars) to view option details, if applicable, for the product. This report can be exported to an Excel spreadsheet.

 Options by revenue

This report displays the option level data for the filtered time period. Hover your mouse over any section in a pie chart to view percentages. Also, click on a section to separate it from the pie chart. Click on it a second time to return it to its original position in the chart.

More detailed information appears to the right, including item, SKU, number of items, price, tax, shipping, and totals. This report can be exported to an Excel spreadsheet.

 Products by conversion high

This report displays the best converting products for the filtered time period. Detailed information includes SKU, views, orders, merchandise totals, tax, shipping, and order totals. Click View Details (binoculars) to view option details, if applicable, for the product. This report can be exported to an Excel spreadsheet.

 Products by conversion low

This report displays the least converting products (lowest conversions) for the filtered time period. Detailed information includes SKU, views, orders, merchandise totals, tax, shipping, and order totals. Click View Details (binoculars) to view option details, if applicable, for the product. This report can be exported to an Excel spreadsheet.

To create a Top Products report

  1. In Analytics, select Product Reports > Top Products.
  2. Select report search criteria:
    2. If Tier Groups are set up on your site, you can select to filter your search by specific group, all groups, all customers, and so on.
    3. Choose an Affiliate if applicable for your site.

    4. Choose an Order Status: search for all, active and complete, or complete only.

       Choose Report Options: You can filter your product search by revenue, product views, specify a product, product options by revenue, or products by conversions (high or low) as explained above.

  3. Click Search.

  4. For any of the reports generated, click on View Details button to view option details for the product in a pop-up window. You can also export a report to Excel from this view.
