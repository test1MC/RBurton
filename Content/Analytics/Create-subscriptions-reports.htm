Creating Subscriptions reports

Use this report to check on subscription trends and plan your free trials accordingly. You can search for subscriptions by their Requested Date, Scheduled Date and Completed Date.

The report displays:

  • Days—refers to the number of days from the day the order was placed. This way, if you have, for example, a 30-day money back guarantee, you can see whether the number of days falls within the allotted time frame. If the order is canceled, “days” signifies the number of days from when the order was placed until it was canceled.
  • Card Processed—relates to the number of attempts that were made to charge the card.

To create a Subscriptions report

  1. In Analytics, select Order Reports > Subscriptions.

  2. Use the Report Type list to select the level of detail for this report.

    Note: Choosing a Detailed report type causes more filters to appear, allowing you to perform a more precise search. If you select Detailed report type:

    2. Choose a Status from the list, for example, cancelled, declined and so on, or select All.
  3. Click the Search button.
