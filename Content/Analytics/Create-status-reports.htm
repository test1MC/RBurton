Creating Order Status reports

The Order Status report provides a breakdown of the state of your orders over a specific date range. Use it to keep track of orders' statuses in the completion process, and analyze any inconsistencies or spikes in returns, back-orders, or incomplete orders. Then, see if a change to your ordering process improves the status report results over time.

A pie chart shows item breakdown by status, as well as a table showing this information along with counts.

To create an Order Status report

  1. In Analytics, select Order Reports > Status.
  3. If Tier Groups are set up on your site, you can select to filter your search by specific group, all groups, all customers, and so on.
  4. Click the Search button.
  5. Click on any chart segment to display a table showing all items with that status, along with additional information related to each item.
