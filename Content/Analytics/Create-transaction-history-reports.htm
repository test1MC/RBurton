Creating Transaction History reports

Use this report to reconcile your order transactions against the transaction history information provided by the payment processor. This allows you to verify that you have been paid on each transaction. If a transaction has been voided and/or credited, these orders are displayed, and are associated with a negative dollar amount. You can also use the report to settle or synchronize Authorize.net orders.

To create a Transaction History report

  1. In Analytics, select Order Reports > Transaction History.
  3. Click the clock icons to choose a time frame for the report.
  5. From the Transaction Type list, select the type of transactions you want to see (settled, credit, and so on).
  6. Click the Search button.
  7. To export the report to Excel for reporting or printing, scroll to the bottom of the report, and click the Export to Excel button .

Settling or synchronizing Authorize.net orders

Use can use the transaction history report to search for credit card transactions. It shows all transactions that occurred with the credit card processor for the orders found within the selected date range. Note that:

  • It provides a unique line displays for authorizations, captures, credits and settlements.
  • Settlements are currently only available for Authorize.net. When you import an Authorize.net end of day settlement report, you can easily reconcile all settled transactions against those transactions still waiting to be settled.
  • This enhancement will help accounting departments more easily close their books at end of month.

To upload the Authorize.net settlement report

  1. Scroll to the bottom of the transaction history report, and click the Import Settlement File button.

  2. Select the file, and click the Save button.

  3. Run the report and it shows all orders over the specified date range.

    The Transaction Type column indicates what type of transaction is indicated in the row. There are four options:

    1. Authorized
    2. Captured
    3. Settled
    4. Credited
