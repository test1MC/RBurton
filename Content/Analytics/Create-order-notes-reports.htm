Creating Order Notes reports

Each time an order status changes, notes are automatically added to the Order History. Notes can also be manually added as needed.

Use the Order Notes report to view order notes recorded during a specific date range, and by the user who changed an order.

To create an Order Notes report
  1. In Analytics, select Order Reports > Order Notes.
  2. Use the lists to select a date range for notes you want displayed.
  3. In the --Note Written By list, either select the user whose notes you want to see, or to see notes from all users, leave as is.
  4. Click the Search button.
 To export order notes
  1. In Content, select Import/Export > Order Notes.
  2. Click the Export button to export order notes between a selected range and user.

Note: Always do an export to ensure you are using the most recent spreadsheet version. You can also use it as a guide for proper format for each column and row, as well as which fields are required.

 To import order notes

  1. In Content, select Import/Export > Order Notes.

  2. Click the Browse button to locate the file in a local source. Click the Import button to import the new spreadsheet
