Creating By Location reports

Use this report to determine where your site traffic is coming from, US or worldwide. The report shows where visitors are originating, including Country, Region, and City, and the Percentage of Visitors per location.

To create a By Location report

  1. In Analytics, select Traffic Reports > By Location.
  2. Use the lists to select a Begin Date.and End Date. Click the Clock button to select standard times from 12AM – 11PM for report begin and end dates, or type the time range you want.
  3. From the Reporting Type list, choose whether to create a report based on US states, or that includes worldwide traffic.

  4. Select whether to Exclude IP Addresses; this excludes the actions of users from specific IP addresses in the report results.

    Note: To manage which IP addresses are excluded, navigate to Users > Website Settings > General > General Information.

  5. Click Search.
