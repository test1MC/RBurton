Creating Sales By User reports

Use this report to see sales information by user. The report displays a bar graph and a data table showing:

  • Salesperson—also showing web orders as a “Salesperson”
  • Order Count—the total order count,
  • Item Count —total items from those orders.
  • Average Items (average number of items per order sold by the salesperson)
