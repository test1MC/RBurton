Creating Historical Affiliate Tracking reports

Use this report to see how many marketing channels customers accessed before they place an order. If a customer touches an email, a search, or affiliates all within minutes of ordering, these various channels all get credit for the order. Aptos Digital Commerce tracks a customer in a 'multi-touch' manner. If a customer comes to your site from a specific affiliate source, and that source changes along the way multiple times, all this information is captured. This provides better tracking information than one which only includes the last touch a customer has with an affiliate. If a customer searches using the in-site search, Digital Commerce also captures that information.

This feature is available to all clients. However, if you attempt to use this report and do not see any data, please submit a support ticket, and the Support Team will enable this functionality for your site.

To create a Historical Affiliate Tracking report

  1. In Analytics, select Channel Reports > Historical Affiliate Tracking.
  3. Click the Affiliate drop-down and choose either all affiliates, or a specific individual affiliate or customer.
  4. Click the Affiliate Group list, and choose the group appropriate for your search.
  5. Click Search.

About information the report provides

This report shows the affiliates that were given credit for various orders, and provides information on:

  • Token - the tokens used on the affiliate page.
  • Customer Name - the name of the customer.
  • Affiliate - this column shows the actual affiliate.
  • Affiliate Group - the group the affiliate belongs to.
  • Affiliate ID - ID number assigned to the affiliate.
  • Visit Date - date the customer visited the page.
  • Order # - the order number.
  • Subtotal - subtotal of the order.
  • Shipping - shipping price charged to the order.
  • Tax - tax total charged to the order.
  • Discount - discount amount, if any, applied to the order.

  • Promo Code - if applicable, the promo code used on the order.
  • Page Name - the page name.
  • URL Variables - variable for the URL for the affiliate.
  • Insite Search - Insite search terms applied to the affiliate.
  • Search Keyword - keyword terms applied to the affiliate.
