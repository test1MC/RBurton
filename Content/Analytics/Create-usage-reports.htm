Creating Usage reports

Use this report to keep track of your Site Manager users' activities. This report shows settings or price changes users have made in the Site Manager, and provides an audit trail to their page visits.

The report displays activity information in five areas:

Website Settings

  • User
  • Field
  • Old Value
  • New Value

To create a Usage report

  1. In Analytics,select User Reports > Usage.
  3. From the User list, select the user you want to audit, or do not change to view all users.
  4. Click Search .

  5. Click the View Details link to open up an Ajax window with specific information on the usage for that session.
