Creating Trends reports

Use the Trends traffic report to see traffic driven to your site. You can create the report for a specific time period, and filter by specific affiliate or group, so you can see emails compared to channels, and so on. The graph compares traffic, order revenue and mobile traffic.

Note: The Total Visitors column reflects all affiliate hits by spiders, bots crawling on the site, and so on. Actual Visitors includes only the number of affiliate hits by people who visited your site.

To create a Trends traffic report

  1. In Analytics, select Traffic Reports > Trends.
  2. From the Show traffic for list, choose the period for which you wish to create the report:
    1. Last 30 Days
    2. Year– also select the specific year for which you would like to display results from the new list that appears here. The system default is the current year. You can choose a different year; results will be grouped by month.
    3. All Years – displays the year over year trend for all years your site has been on Aptos.
  3. From the Affiliate Group list, choose to filter by a specific affiliate or group, or by all, if applicable.

  4. Select whether to Exclude IP Addresses; this excludes the actions of users from specific IP addresses in the report results.

    Note: To manage which IP addresses are excluded, navigate to Users > Website Settings > General > General Information.

  5. ​​Click the Search button.

    The Traffic Trend for Year to Date by Month graph displays the date range, visitors, mobile visitors and revenue. The raw data is displayed under the chart.

  6. ​Click any view option below the graph—Visits, Dollars, Mobile—to remove that data from the chart. Click again to return it to the chart.
