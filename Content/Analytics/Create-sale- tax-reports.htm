Creating Sales Tax reports

Use this report for tax reporting and credit card reconciliation. It filters sales data using either the order date or the credit card capture date. It only returns data on shipped items with the item status of "Item Shipped - Tracking Sent" and "Item Shipped - Waiting for Tracking."

Report results do not change based on a return order unless the return occurred in the same month, because it uses the return dates for refunds on credit cards.

For example, if sales for January were $300,000 with $20,000 worth of returns for orders taken in January, this report would show $280,000 in sales for January and would not change past the month closing. Also, if a return took place in January for a December order, the return date refund on the credit card would be for January and not December.

To create a Sales Tax report

  1. In Analytics, select Order Reports > Sales Tax.
  3. If Tier Groups are set up on your site, you can select to filter your search by specific group, all groups, all customers, and so on.
  4.  Choose an Affiliate if applicable for your site.
  5. Select a Payment Type on which to filter, or choose nothing to report all.
  6. Select a State; you can select all states, or the default state for your shipments.
  7. Select a Report Type.
  8. From the Date to Use list, select to filter on either the Order Date or Capture Date.
  9. Click the Search button.
