Customizing Product reports

Use to create a customized product report based on criteria you choose. Use an existing custom report view, or create and save a new one.

To create a customized Product report from an existing view
  1. In Analytics, select Product Reports > Customize.
  2. To use a previously saved view, click the Select Existing View list, and select the view you want to use.
  3. If applicable, choose the query level for your report, either Order Level, or Item Level.
  4. Click theSearch icon to run a query for your report.
  5. In the Result Options area at the bottom of the page, click the View icon to see the customized report in a pop-up window. Scroll to the right to see all information it contains.
  6. To export the report to an Excel spreadsheet, close the pop-up window, and in the Result Options area, click the Export to Excel button. Make sure your browser is not configured to block pop-ups.

  7. To edit the report, click the Edit button.

 To create and save a new customized report view
  1. In Analytics, select Product Reports > Customize.

  2. Click Add Search Condition and enter the Field, Operation, and Compare To information.

    Note: To add another row of search conditions, select And or Or at the end of the row, and click Add Search Condition.

  3. Click theSearch icon to run a query for your report.

  4. To save this report view, in the Result Options area that appears at the bottom of the screen, enter a unique name for it in the Save View As field, and click Save.
