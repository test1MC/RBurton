Creating Commission reports

Use this report to view the progress of your sales representatives in a detailed or summary format.

Commission Report: detailed

Commission Report: summary

To create a Commission report

  1. In Analytics, select User Reports > Commission Report.
  3. Select whether you want to run a Detailed or Summary report.
  4. Click the Select All link above the Salesperson list, or click check boxes to select one or more salespeople from the list.
  5. Click the Search button.
  6. Click Export to Excel button for a printable version of the commission report organized by Salesperson.
