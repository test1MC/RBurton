Creating Sales Trends reports

Use this report to stay on top of your sales trends from the last month, year, or all years. Information is presented in five graphs, showing trends for based on the time period you select:

  • Total Orders (Qty),
  • Total Orders ($),
  • Average Order Value,
  • Total Items Sold,
  • Order Credits ($).

The report also provides a data table of sales trends.

To create a Sales Trends report

  1. In Analytics, select Order Reports > Sales Trends.
  2. Use the Date Range list to select a the range you're interested in—last 30 days, current year, and so on.
  3. Use the Order Status list and choose to search Active or Complete, Complete only, or All Orders.
  4. Use the Order Type list to choose to search Core Site Orders, Mobile Orders, or All Orders.
  5. If Tier Groups are set up on your site, you can select to filter your search by specific group, all groups, all customers, and so on.
  6. Use the Salesperson list to select one or more agents, or click the link to Select All.

  7. Select whether to Exclude IP Addresses; this excludes the actions of users from specific IP addresses in the report results.

    Note: To manage which IP addresses are excluded, navigate to Users > Website Settings > General > General Information.

  8. Click the Search button.
