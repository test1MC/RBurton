Creating Returns Without Credit reports

Use this report to identify these items and give credit where appropriate.

This report displays items having an 'Item Return Received' status that do not yet have an associated credit record. It also displays items where credit records exist but whose date stamps are older than the Item_Return_Date.

To create a Returns Without Credit report

  1. In Analytics, select Order Reports > Returns with No Credit.
  2. Click the appropriate button to filter by Return Date or Order Date.
  3. Use the Payment Type list to filter by specific payments type, or to show results for all payments; default is All Payments.
  4. Use the Return Reason list to filter by the appropriate reason or all reasons; default is All Reasons.
  5. Click Search.

  6. Click on an order number to go to View Order tab, which shows order details.

    Note: The order number shown here will be either the Aptos Order Number or External Order ID depending on the state of the website setting 'In Site Manager, display External Order IDs instead of Aptos Order Numbers. Website Settings are in Users > Website Settings.
