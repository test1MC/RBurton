Creating 404 / Detailed Traffic reports

The 404An error message displayed by a browser indicating that an Internet address cannot be found. / Detailed Traffic report is a comprehensive and detailed report on your site and includes everything from activity statistics down to the hour of day, daily page statistics, directory access information, file type access information, a list of top referring sites, daily search keywords, and more. Aptos pulls the detail report for clients on a monthly basis.

The 404 / detailed traffic page is divided into two sections: The 404 Report and the Detailed Traffic Report. The 404 Report is displayed by default.

404 report

The 404 report displays a bar graph and a compiled list of all of the pages that caused a redirect to the 404 page: that is, the page displayed when a page fails to load. The graph displays either the top nine results if sorted by number of hits, or the first nine results if sorted alphabetically. The number of results shown in the list is defined by the value selected in the # of Results menu.

To create a 404 / Detailed Traffic report

  1. In Analytics, select Traffic Reports > 404/Detailed Traffic.

  3. From the Report Type list, select either 404 Report or Detailed PDF .

    The 404 report is displayed on this page; the Detailed traffic report provides a PDF link you can open and download.

  4. From the # of Results list, select the maximum number of results you want displayed in the report.

  5. Click the Search button.

  6. If you chose the 404 report:

    1. Click on the 404 Page link to sort the list alphabetically and update the graph.

    2. Click the 404 Hits link to sort the list by the number of times a page load failure caused a redirect to the 404 page, and to update the graph.

  7. If you chose the Detailed Traffic report, click the report.pdf link to download and open the file.
