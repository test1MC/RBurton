Creating Custom Order reports

Create your own customized reports with the search parameters you want. Is there a special promotion that you just ran? Create a report tied to the affiliate code and see how your sales were affected. Do you want to see how many of your customers are using an alternate payment method? Create a custom report for the payment type.

To create a Custom Order report

  1. In Analytics, select Order Reports > Customize.
  2. Choose the settings for the report:
    1. Click the Select Existing View list to use an existing saved view, or
    2. Add Search Condition: click this area to add a new condition and enter the Field, Operation, Compare To, and And/Or information. Repeat to add as many search conditions as you need.
  3. Select condition:
    1. Order Level
    2. Order Level with Transaction History
    3. Item Level; in addition to the other columns in the report, the UPC at the product and option level is available on the Item Level export.
  4. Click Search to run a query on your order report.
  5. To save the view so you can re-run it later, enter a name for the view in Save View As, and click Save.
  6. Click View to see the customized report in a pop-up window.

  7. To manage the orders that result from the custom report, click the Manage button , all orders that fall within the specified parameters are displayed on the Commerce> Orders > View page for you to edit, batch process, or to perform whatever function you wish.
