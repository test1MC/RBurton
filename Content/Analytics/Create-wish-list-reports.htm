Creating Wish List reports

Use this report to manage your customers' wish lists. You can use wish list data to send targeted emails or promotions to customers.

If customers have added products to their wish lists within the range you specify for the report, they appear in the search results. If none have added wish list items within that date range, no results appear.

To create a Wish List report

  1. In Analytics, select Traffic Reports > Wishlist.

  3. Click the Search button.
