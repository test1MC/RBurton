Creating Daily Trends reports

The Daily Trends report displays sales data for the date range you choose. You can filter to narrow the search results. Information is displayed in a chart and table with Average Daily Orders, Average Daily Item Amount, Average Daily Total shown under the search filters.

Depending on the date range, you will see statistics on:

  • total sales per day
  • total orders per day
  • average order value

To create a Daily Trends report

  1. In Analytics, select Order Reports > Daily Trends.
  3. Select the Order Status:
    1. All Orders
    2. Only Completed Orders
    3. Shipped and Completed Orders
  1. You can also sort by Tier Groups, Heard From, State, City, DMA Code and Area Code.
  2. Click Search.
