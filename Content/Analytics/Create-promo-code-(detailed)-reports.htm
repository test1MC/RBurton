Creating Promo Code (Detailed) reports

Use this detailed report to see the status and success of your promotional campaigns. Find answers to questions such as:

  • Did customers in a particular state use a promotion more frequently?
  • Did a Twitter campaign do better than an email campaign?
  • Did one promo code bring in fewer customers, but produce higher order totals and more overall revenue as a result?

Analyze trends in this report and refine your marketing strategies accordingly.

To create a Promo Code (Detailed) report

  1. In Analytics, select Channel Reports > Promo Code (Detailed).
  3. Click Search.
  4. To export the report to an Excel spreadsheet for reporting or printing, click the Export To Excel button .
