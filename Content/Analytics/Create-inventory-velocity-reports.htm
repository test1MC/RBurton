Creating Inventory Velocity reports

The Inventory Velocity report provides information on recent sales of each SKU, as well as an estimate of how long the existing quantity will last. This report is useful for identifying those SKUs which are "fast moving" and at risk of selling out quickly. Use this report to identify SKUs that need to be reordered, as well as SKUs that are slow moving and may need to be discounted or written off. Data in the report includes:

  • SKU

  • Part/Style #

  • Alt Part #

  • Name (Product Name)

  • Inventory Count

  • Qty Sold

  • Days of Inventory

  • Stock Status

  • Selling Price

  • On Sale Price

To create an Inventory Velocity report

  1. In Analytics, select Product Reports > Inventory Velocity.
  3. From the Status list, choose to create your report based on:
    1. All Orders
    2. Only Completed Orders
    3. Shipped and Completed Orders
  4. Select whether to Include All Inventory in the report.
  5. Click Search.
