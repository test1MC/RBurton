Creating Saved Orders reports

Use this report to view open proposals which are essentially non-ordered, created orders, deleted orders, and converted orders. These are proposals that successfully become monetized orders.

Results are displayed in a pie chart showing converted, deleted, and open proposals; details for each of these proposal types are shown in tables below the pie chart.

To create a Saved Orders report

  1. In Analytics, select Order Reports > Saved Orders.
  4. Use the Salesperson list to select one or more agent, or leave all unchecked to create a report for all agents.
  5. Click the Search button.
