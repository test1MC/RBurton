Creating Payment Types reports

Run the Payments Types report to view a breakdown of data based on credit card types, alternate payment methods, and PO terms.

To create a Payment Types report

  1. In Analytics, select Order Reports > Payments.
  2. Select a date range using the From Date and Thru Date fields.
  3. Select an Affiliate, if applicable.
  4. Filter by Order Status by choosing one of these values:

    • All: all orders including cancels.

    • Active and Complete: all orders that are not in a canceled nor declined status. Orders that are shipped or are going to ship.

    • Complete Only: orders in a shipped or completed status.

  5. Click Search. When the search is complete, the statistics appear on this page. When an order is returned, it is not subtracted from the All order status.

  6. Click Export to export a spreadsheet to Excel.

Viewing Payment Types details

Hover over chart pieces to see the percentage of the order total for each payment type.
