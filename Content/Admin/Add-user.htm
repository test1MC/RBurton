Adding a new user

Use this procedure to add a new EOM user account.

  1. In Users, click ADD USER. The Add User window opens.

  2. Enter the user information here.

    Only the Full Name, Username, and Email Address fields are required to add a new user; the others are optional.

  3. Select the check box to designate which sites to the user has access. If sites are tied together, you can set up the user once, and grant them access to all of those sites here. Keep this in mind if you ever need to restrict access for any reason. Aptos never deletes information from our database in order to meet compliance regulations.

  4. Click the Add button + to add the user.

    sends an email to the user's email address, prompting them to change their password.
