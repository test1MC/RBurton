Importing and exporting user accounts

Use this feature to import and export user accounts using an Excel spreadsheet.

You can export a spreadsheet with all existing user accounts to use as a template when importing.

When updating by spreadsheet, you should always export first to ensure you are using the most recent spreadsheet version. This will also guide you on the proper format for each column and row as well as which fields are required.

To export user accounts

  1. In Users, click IMPORT/EXPORT USERS. The Import/Export Users window opens.

  2. Click the Export to Excel button . This generates an Excel spreadsheet, and a link to it. Click this link to open, modify, and or save it to your hard drive.

Importing user accounts

Excel file headings

The Excel file you import must have these headings:

Heading Description

UserID

A unique ID generated by the platform used by the system to identify the user's record in the database.

Name

The user's first and last name.

Email

The user's email address; used to identify a user in the system. This field is required to generate a new user account.

Phone

The user's phone number.

Address

The user's street address.

City

The user's city of residence

State

The user's state of residence.

Zip

The zip code of the user's residence.

UserName

Can be a unique name for display purposes and can be different then the user's first and last name, as well as their email address.

orderAdmin

True/False value; if False, prevents a CSR from editing a product's pricing from the retail order screen. Set to True to allow them to modify product pricing.

active

True or False value; if set to True makes a user's account active in the site manager. False prevents a user from logging into the site manager.

Password

Unique password a user will use to log-in to their site manager account.

avatar

The image path for the uploaded image that will represent a user in the site manager.
To import user accounts

  1. In Users, click IMPORT/EXPORT USERS. The Import/Export Users window opens.

  1. Click Browse, find the Excel spreadsheet you want to import, and click Open.
  2. Click the Import button .

    User accounts are automatically uploaded to the server.
