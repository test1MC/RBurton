About Infobase tables

Infobase is a transactional database used by Merchandising to process data. Infobase contains tables that store the results of activity from Merchandising transactions and sends this data to other Aptos applications and external systems.

Infobase tables

On Order Purchase orders, receipts and price changes affecting units, cost and retail on order
Inventory Transactions affecting units, cost and retail inventory. Inventory is tracked by transaction code for receipts and RTVs
Prices Past, current and future prices and price statuses including any mix and match codes
Allocated Quantities Transactions affecting allocated quantities of on order and/or on hand merchandise

An important system administration responsibility is to monitor the movement of data through to ensure that transactions are correctly processed, and that the correct values are stored in Infobase tables. Because Infobase doesn't have a user interface, you verify values by comparing the expected result of a transaction with its actual outcome.

This section contains the information you need to calculate the expected result of a transaction. The transaction table shows how the transaction types update the Infobase tables.
