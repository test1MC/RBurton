Editing a user's information

Edit a user's account information, including their name, address, email address and so on.

To edit a user's account information
  1. In Users, find the user whose account you want to edit.
  2. Click the Edit link in the user profile, and you'll be taken to the Add User pop-up window.
  3. Make any changes to the user's information as needed.
  4. Click Save.
 To assign or change a user's avatar image
  1. In Users, find the user whose account you want to edit.

  2. Click the Change Picture link in the user profile, and the Change Picture pop-up window opens.

  3. Choose the picture the user wants to use; either:
    1. Select an image from those presented in the window, or
    2. Select the Use Facebook Profile Image check box, or
    3. Click Browse to find and select an image saved on your computer.
  4. Click Save, and close the window.
 To disable a user account
Disabling a user account prevents that user from logging into the site manager. It does not delete the user account.
  1. In Users, find the user whose account you want to disable.
  2. Click the Disable link in the user profile.
 To enable a disabled user account

  1. In Users,de-select the Only Show Active Results check box. This ensures that disabled accounts are displayed along with enabled ones.

  2. Find the user whose account you want to enable.
  3. Click the Enable link in the user profile.
