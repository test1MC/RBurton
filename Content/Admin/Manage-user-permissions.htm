Managing user permissions

Once you have set up a user, you can assign permissions to ensure they only have access to what they need in .

To assign permissions
  1. In Users, click PERMISSIONS.

  2. Select the Site and User from the list menus.

  3. Ensure the Site Manager Permissions radio button is selected, and click the Search button to continue
  4. Set individual permissions listed as needed; you can also use the Select All and Remove All links at the top of the page.
  5. Click Save.

About Admin User reports

This report provides a high level overview of all user permissions set in Site Manager. It gives you a general idea of the level of access given to each user. To access the report, go to Analytics > User Reports > Permissions. See Creating Permissions reports for more information

Tracking users

A field in the database tracks when a new Admin User has been added to the Site Manager. When a new admin user is created, the date the user was added is captured.

When changes are made to an Admin User's permissions, the date the permissions were modified is captured. This is useful for tracking and reporting purposes
