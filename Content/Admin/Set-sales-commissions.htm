Setting sales commissions

If a user added to the site manager is also a sales representative, you can assign a commission structure to that user.

To set commissions for a user

  1. In Users, click SALES COMMISSIONS. The Sales Commissions window opens.

  2. Select the salesperson from the list; the current commission structure for that sales person is displayed. If none are defined for this salesperson, click the Add First Commission to Selected Salesperson link.
  3. Set commission bracket levels in the Low Sales Amount and High Dollar Amount fields. These determine the dollar amounts between which the commission percentage applies.
  4. Enter in the Commission Percentage. When the salesperson makes a sale that falls between the low and high sales amounts, they will be credited with the related commission.
  5. To add more commission brackets, click the Add New Record link.
  6. Click Save.
