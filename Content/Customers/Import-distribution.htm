Importing distributions

  1. Create an XML text file that conforms to the file layout.
  2. Run the Merchandising.Import XML Documents Services Administration job. For information on running jobs, see the Job Management Reference.

EntityAction

Attribute

String

“Freeze"

“FreezeHeader” “Unfreeze"

“Modify”

"Complete"

"Cancel"

No

"Freeze" is issued by WM to take over control of a distribution. This will either freeze entire document or just some of the distribution lines.

“FreezeHeader” is issued by WM and it will only freeze the distribution header while leaving all distribution lines in an unfrozen status, WM uses this action to alert the allocator that WM picking will begin soon but has not happen yet.

"Unfreeze" is issued by WM to release the distribution back to A&R as an allocator needs to adjust the quantities within A&R.

“Modify” is issued by WM when it adjusts the distribution quantities.

"Complete" is issued by WM when shipping against the distribution or a distribution line has been completed.

"Freeze", "Unfreeze" and "Complete" can be applied against individual lines or for the entire distribution.

"Cancel" is issued by WM when it needs to cancel an entire distribution document. Outstanding allocated quantities will be backed out from Infobase.

Create, Submit, Approve, Reject, Release and Reinstate are also valid import actions but they typically happen within the A&R application and are not applicable for import integration purposes (unless the distribution was created outside of A&R).

XML examples

To freeze the document

Works the same way for a document complete, cancel or unfreeze.

<?xml version="1.0"?>

<Distributions>

<Distribution CompanyId="1" EntityAction="Freeze" action="update">

<Number>001271</Number>

</Distribution>

</Distributions>
To freeze a distribution line

Works the same way for a document complete, cancel or unfreeze.

<?xml version="1.0"?>

<Distributions>

<Distribution CompanyId="1" EntityAction="Freeze" action="update">

<Number>001271</Number>

<DistributionLines>

<DistributionLine action="update">

<StyleCode>100000079</StyleCode>

<ColorCode>030</ColorCode>

</DistributionLine>

</DistributionLines>

</Distribution>

</Distributions>
To freeze a distribution header

Works the same way for a line unfreeze, line complete.

<?xml version="1.0"?>

<Distributions>

<Distribution CompanyId="111" EntityAction="FreezeHeader" action="update">

<Number>006905</Number>

</Distribution>

</Distributions>
To adjust distributed quantities

<?xml version="1.0"?>

<Distributions>

<Distribution CompanyId="1" EntityAction="Modify" action="update">

<Number>001271</Number>

<DistributionDetails>

<DistributionDetail action="update">

<Quantity>30</Quantity>

<LocationCode>1100</LocationCode>

<PackCode />

<StyleCode>100000079</StyleCode>

<ColorCode>030</ColorCode>

<UPCNumber>400000004853</UPCNumber>

</DistributionDetail>

</DistributionDetails>

</Distribution>

</Distributions>
