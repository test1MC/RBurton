Importing and exporting customer data

EOM gives clients who have existing clientele the ability to import customers into the site manager's CRM. This function is not intended to be a historical order import; rather, it enables our clients to bring in existing customers and user-specific data.

Use this feature to import and/or export customer information, account questions, and address book entries. It also allows for exporting of customer information with various filters.

When updating by spreadsheet, you should always export first to ensure you are using the most recent spreadsheet version. This will also guide you on the proper format for each column and row as well as which fields are required.

To export customer data
When exporting in order to generate an import template, it is recommended that you select both the Include Account Questions and the Include Address Book Entries check boxes. This way all possible fields will be included when importing, and discrepancies will be eliminated.
  1. In CRM, select Customers > Import/Export.
  2. In the Export Customers area of the screen, select the options you want:

    1. Create Date From/Create Date Thru – Define a date range from which you wish to pull customer information. This range will only pull customer information for customers that were added/created during the date range. Customer information added outside of this date range will not be pulled.

    2. Create Email/Password Reset File – Useful for importing customers from another database into the Site Manager. Once your customer information has been imported, select this option to export the customer list. This creates a file containing email addresses and encrypted unique URLs for each individual customer. The "Reset URL" strings can be used to invite your customers to reset their passwords on your new site. (Additional Wiring Required).

    3. Include Account Questions – Populate the Excel spreadsheet with two new fields: Q_Text and Answer_Text. This a custom feature. (Additional Wiring Required).

    4. Include Address Book Entries – Generates a new row in the spreadsheet for each unique address in the address book. If there is more than one row for a particular customer, the first row is from their record (billing), while the rest of the information will be from the address table. This will include all of each customer's addresses, including the billing and shipping addresses. If the check box is not marked, the address returned will be the billing address.

      To export all customers with no address book entries--no duplicate customer information--make certain no checkboxes are selected

  3. Click the Export to Excel button .

    This generates an Excel file and a link to it that appears in the Available Exports to Download area of the page.

    If the Create Email/Reset File was selected, the CSVcomma separated value customer password reset file will be added to the list of available files. The reset file is an XML document that generates an unique reset URL that can be used manually or sent via email to the customer if wired to do so.

  4. Click the spreadsheet link, add or update customer information in it, and save the spreadsheet to a local drive.

To import customer data
  1. In CRM, select Customers > Import/Export.
  1. Click Browse, find the Excel spreadsheet you want to import, and click Open.

  2. Click the Import button .

    Importing customer information automatically updates the customer database with new information and creates additional records for new customers. If you prefer to do this manually, you can add customers one by one on the CRM > Customers > Add/Edit page. See Adding and editing customers for more information.
