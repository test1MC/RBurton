Importing min max information

  • Create an XML file that conforms to the file layout.
  • Initiate the import from the A&R Table Maintenance function for Min/Max Profiles. For information on jobs, see the Allocation & Replenishment User Guide and the Job Management Reference.

File layout

XSD definition

The XSD file is installed with Merchandising and is called Min Max Profile Import.xsd.

Notes

If loading a record that already exists, the system will overlay the existing record with the new imported record.
