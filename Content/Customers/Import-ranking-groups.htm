Importing ranking groups

  1. Create a tab-delimited text file that conforms to the file layout. The import also supports a comma-delimited text file and an XML file. Contact Aptos Client Care for details about these options.
  2. Initiate the import from the A&R Table Maintenance for Ranking Groups. The import can be done on-line or scheduled.
    If scheduled, set up the schedule in A&R and then define the import job from Services Administration. For information about scheduling jobs, see Scheduling in Allocation & Replenishment and the Job Management Reference.
File layout

Field #

Name

Data Type

Mand

Valid Values

  

1

Ranking group code

  

Yes

  

User defined name that identifies the ranking which can be based on a lists of styles, list of classes, and so on. In this case, the ranking is NOT linked to any specific merchandise group or style.

2

Merchandise group code

 

*

  

Mandatory if creating a ranking group for a merchandise group.

Indicates a ranking group will be created for specific merchandise group from the main merchandise hierarchy.

Must adhere to the merchandise group mask defined in Product.

3

Style code

 

*

 

Mandatory if creating a ranking group for a style.

Indicates a ranking group will be created for specific style.

Must adhere to style code mask defined in EDM parameters.

4

Grade code

10 AN

Yes

 

Grade code assigned to the ranking group.

5

Minimum

Integer

No

Numeric value >=0

Defaults to 0 if not provided.

The Minimum unit value associated to each Grade for the designated Ranking Group. The Minimum value can be used by the distribution tools to assign units to a location.

6

Maximum

integer

No

Numeric value >=0

Defaults to 0 if not provided.

The Maximum unit value associated to each Grade for the designated Ranking Group. The Maximum value can be used by the distribution tools to assign units to a location.

7

Weight

Decimal (4,2)

No

>=0.00

Defaults to 0.00 if not provided.

The weight factor value associated to each Grade for the designated Ranking Group. The weight factor can be used by the distribution tools to assign units to a location.

8

Location code

 

Yes

 

Location assigned to each grade in that ranking group.

Must adhere to the location code mask defined in EDM parameters.
