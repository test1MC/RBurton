Importing customers

  1. Create a tab-delimited text file that conforms to the file layout.
  2. Run Pipeline segment 16031 - Text File to IM Import Tables. This segment loads the ASN text file into ASN import tables.
  3. Run Pipeline segment 19010 IM Import to IM - ASN Import. This segment loads the ASNs from the ASN import tables into IM, creating the actual ASN document and potentially auto-generate PO receipts off the ASNs.

You can import ASNs using an XML file and the Merchandising.Import XML Documents Services Administration job.
