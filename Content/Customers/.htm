08

Primary Flag

*

  

Y/N

If True then all other vendors for this style have this flag set to false. If this field is empty, it will default to False. Y/N.

Notes

A record is not expected for the Primary Vendor. This information is provided through the STYLE (RS) interface file.

A ‘M’odify record to switch the Primary Flag to False, will reject since the interface has no way of knowing which Vendor Style should be set to be the Primary one. Therefore, to change the Primary Vendor of a style, an ‘A’dd or a ‘M’odify record with the Primary Flag set to True, for the new Primary Vendor Style, should be interfaced. The existing Primary Vendor will automatically have this flag switched to False.
