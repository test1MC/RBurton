Importing ASNs using XML

  1. Create an XML text file that conforms to the XML file layout.
  2. Run the Merchandising.Import XML Documents Services Administration job. For information about running jobs, see the Job Management Reference.

You can also import ASNs using a tab-delimited file and Pipeline.

XML file layout

XSD definition

The XSD file is installed with Merchandising and is called ASN Import.xsd.

XML examples

To create an ASN

<?xml version="1.0" ?>

<ASN Action="add" CompanyId="15">

<ASNStatus>1</ASNStatus>

<ExpectedReceiptDate>2010-08-20T00:00:00</ExpectedReceiptDate>

<ShipmentRefNo>PO-102743</ShipmentRefNo>

<VendorCode>100001</VendorCode>
<POLocations> <POLocation Action="add"> <LocationCode>0011</LocationCode> <PONo>102743</PONo> <SKUs> <SKU Action="add"> <UPCNo>400000029719</UPCNo> <UnitsSent>100</UnitsSent> </SKU> </SKUs> </POLocation></POLocations></ASN>
To modify ASN status

This updates ASN status from preliminary to published.

<?xml version="1.0" ?><ASN Action="update" CompanyId="15"><DocumentNo>000340</DocumentNo><ASNStatus>25</ASNStatus></ASN>
To delete an ASN
<?xml version="1.0" ?><ASN Action="delete" CompanyId="15"><DocumentNo>000340</DocumentNo></ASN>
