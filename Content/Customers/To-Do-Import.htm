Importing To Do information

  1. Create an XML text file that conforms to the file layout.
  2. Run the Merchandising.Import XML Documents Services Administration job. For information on running jobs, see the Job Management Reference.

File layout

XSD Definition

The XSD file is installed with Merchandising and is called ToDo Entry Import.xsd.

XML example

To request a new distribution

<?xml version="1.0" encoding="utf-8"?>

<ToDoEntry CompanyId="86" Action="add" >

<POReceiptNumber>101454</POReceiptNumber>

<ToDoEntryDetails>

<ToDoEntryDetail Action="add">

<UPCNumber>400000029047</UPCNumber>

<Units>100</Units>

</ToDoEntryDetail>

<ToDoEntryDetail Action="add">

<UPCNumber>400000029054</UPCNumber>

<Units>100</Units>

</ToDoEntryDetail>
