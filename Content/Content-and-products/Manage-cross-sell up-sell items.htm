Manage cross-sell up-sell items

Cross-sell/up-sell is the practice of selling or suggesting related or complementary products to a customer. Cross-selling is one of the easiest and most effective methods of marketing. For example, a page can suggest related products under You Might Also Like.

Access the cross-sell/up-sell page from either Content > Sub-Categories > Cross-Sell/Up-Sell or Content > Import/Export > Cross-Sell/Up-Sell.

You can create cross-sell groups, but unless you are going to assign cross-sell/up-sell products using the import spreadsheet, you must navigate to Content > Products > Add/Edit to assign cross-sell/up-sell products to a cross-sell group for a product. To learn more, reference Cross-Sell/Up-Sell in the knowledge base article, #1069.

Note: Your site must be wired to accommodate Cross-Sell/Up-Sell functionality. If you are unsure of your site's wiring or want more information, contact your Client Manager.

To add new cross-sell/up-sell groups

  1. In Content, select Sub-Categories > Cross-Sell/Up-Sell.
  2. Click Add (+) to add a new record.
  3. Enter the name of the Cross-Sell/Up-Sell group in the Group Name field, for example: You May Also Like, Related Videos, Complete Your Set, and so on.
  4. To specify the order in which your groups appear, enter a number in the Display Order field.
  5. Scroll to the bottom of the page and click Save.

Exporting and importing cross-sell/up-sell groups

If you need to make many edits to cross-sell/up-sell groups, you can export a spreadsheet that contains all of your predefined cross-sell/up-sell products. You can then update and edit the spreadsheet, and then import it again.

To export cross-sell/up-sell groups
  1. In Content, select Sub-Categories > Cross-Sell/Up-Sell.
  2. If not already expanded, click Import/Export Cross-Sell/Up-Sell Groups to reveal the Import/Export Cross-Sells area.
  3. In the Export Cross-Sells area, click the Export Products by list, and select whether to filter and export by Sub-Categories or Brands.
  4. Select groups to export:
    1. either click Select All to export all, or
    2. click individual groups from the list. Hold the Ctl-key down to select more than one.
  5. To export your selections as a template, select Template Only.

  6. Click the Export button .

    The exported Excel file appears in the Report column at the bottom of the page. Click its link to open it. You can modify the file as needed and save it to your desktop. See Cross-sell/up-sell spreadsheet columns for spreadsheet information.

    Warning: Make sure the cell formatting matches that of the export file. Do not change the column headers, text, or the format of the Excel template.
Cross-sell/up-sell spreadsheet columns
Column Description
Display_Description The item description that will display on your website.
Group_Name The name setup in the system as to which group the item belongs.
Parent_ID This is the Product ID associated with the core product. The core product is the product with which you are offering the additional product.
Parent_PartNo The part number assigned to the core product.
Brand The brand name of the parent product.
Parent_Display_Name The display name of the core product. This also displays the name of the manufacturer.
Child_ID The child Product ID number. The child product is the product you are setting as the cross-sell/up-sell item.
Child_PartNo The part number associated with the Cross-Sell/Up-Sell product. This follows the same rules as Parent_PartNo.
Brand1 The brand name of the child product you set at the Cross-Sell/Up-Sell item.
Child_Display_Name The display name of the child product you set as the Cross-Sell/Up-Sell item. The manufacturer's name is also displayed.
display_order The order within the section you would like the product to display. If all items have a "0" in this column or if this column is left blank, the display will sort alphabetically.
DeleteRecord Enter True in this column to delete the record.
To import cross-sell/up-sell products
  1. In Content, select Sub-Categories > Cross-Sell/Up-Sell.
  2. In the Import Cross-Sells area, click the Browse button next to the Excel File field, locate the spreadsheet on your computer you want to upload, and click Open.
  3. Choose whether to Allow Duplicates.

  4. Click the Import button .

    Depending on the size of the file, it could take a few minutes to upload.
