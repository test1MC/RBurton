Manage comparable specifications

Use comparable specifications to make like-comparisons of your products based on similarities such as height, width, weight, color, fabric, or any other parameters and so on.

Setting up comparable specification groups

The first step in creating comparable specifications is to set up comparable specification groups.

To set up comparable specification groups
  1. In Content, select Sub-Categories > Comparable Specs.
  2. Click to expand the Comparable Specifications Groups area.
  3. Click Add (+) to add a new group. This adds A New Group field in the Group Name column.
  4. Enter the name of each new group to help you organize your comparable specification groups.
  5. Continue to Add (+) enough fields needed for all the comparable specifications.
  6. Click Save before continuing.

Creating comparable specifications

In the Comparable Specifications section you control the comparable attributes for each comparable specification group. You can filter by each comparable specification group if needed.

To create comparable specifications
  1. In Content, select Sub-Categories > Comparable Specs.
  2. In the Comparable Specifications area, click Add (+) as needed to add fields to the Spec Name column
  3. Enter the Spec Name, for example, height, length, color, weight, and so on.
  4. Select a Spec Group with which you would like the comparable attribute to be associated for organizational purposes from the list.
  5. Enter a Display Order value. Entering a value other than '0' overrides the alphabetical listing of the specifications. Your site must be wired accordingly to use this functionality.
  6. You can add images and descriptions:
    1. Click the specification's Image and Description Edit button .
    2. Enter definition text for the specification. You can edit the specification's Name here as well.
    3. To add an image for the specification, click Browse and select and upload an image from your computer.

    4. Click Save to return to the main window.

  7. Edit sub-catagories as needed:
    1. Click the specification's Sub-Categories Edit button .
    2. Select the sub-vategories to which the comparable specification should be attributed.
    3. Click Save to save the selected sub-categories and return to the main window.

  8. Click Save.

    Note: The Save button at the top of the page saves the Comparable Specification Group names. The Save button on the bottom of the page saves all the Comparable Specifications information.
