Importing and exporting locations

Use import/export locations to add, edit and delete physical store information. This is also where you can manage content displayed on your Store Locator page, make use of the Google Maps APIapplication program interface (if wired), and promote store events occurring at physical locations.

The page is organized into four collapsible areas used to edit various aspects of your locations, each accessible by clicking its header:

  • Store Locator Page Content,
  • Locations,
  • Location Groups, The store locator page allows customers to filter location results by location group. For example, if some of your store locations offer rental services and others do not, you could create two different location groups and then assign each store to the appropriate group. This way, the customer can filter the location results to only include the locations that offer rental services.

  • Import Export Locations.

    You should always export first to ensure you are using the most recent spreadsheet version. This will also guide you on the proper format for each column and row as well as which fields are required.

To edit store locator page content
  1. In Content, select Import/Export > Locations.

  2. Click the Store Locator Page Content area header to expand it.

  3. Edit information here as needed. See Store Locator Page Content field descriptions for details.

  4. Click Save.

To edit locations information
  1. In Content, select Import/Export > Locations.

  2. Click the Locations area header to expand it.

  3. You can use search the fields to find a location by name or group and then sort the list by Name, ID, Group, State or Country.

  4. Edit information in the Address column as needed. Note that the unique location ID automatically generated when a new location is added isn't editable.
  5. Edit Information fields as needed:

    1. If applicable, select a Location Group from the list. No Group is selected by default.

    2. If each location has a different Website Address, enter its URL here.

    3. Include an Email Address and Phone number.

    4. Enter the Display Order you would like each location to appear on the Store Locator Page.

  6. Scroll to the bottom of the page and click Save.
 To edit location groups information

The store locator page allows customers to filter location results by location group. For example, if some of your store locations offer rental services and others do not, you could create two different location groups and then assign each store to the appropriate group. This way, the customer can filter the location results to only include the locations that offer rental services.

  1. In Content, select Import/Export > Locations.

  2. Click the Location Groups area header to expand it.

  3. Change Location Group Names as needed, and click Save.
  4. To add location groups:
    1. Enter the number of location Groups you want to add in the Add field, and click New Location Group(s).
    2. Change the default name, A New Group, to the name you want.
    3. Click Save.
  5. To delete a group, select its check box, and click the Delete button at the bottom of the page.
  6. Click Save.
 Importing and exporting locations

If your list of locations is extensive, you can import a spreadsheet of all locations. Aptos always recommends that you export a copy of the most recent spreadsheet to your desktop before updating and importing it.

To export locations
  1. In Content, select Import/Export > Locations.

  2. Click the Import/Export Locations area header to expand it.

  3. Click the Export to Excel button .

    This creates an Excel spreadsheet and a link to it beneath the Import/Export buttons.

  4. Click the spreadsheet link, add or update category information in it, and save the spreadsheet to a local drive.
 To import locations
  1. In Content, select Import/Export > Locations.

  2. Click the Import/Export Locations area header to expand it.

  3. Click the Excel File field's Browse button.
  4. Browse to and select the Excel file you want to import, and click Open.
  5. When the name of the file you chose appears in the Excel file address field, click the Import button to upload new and changed category information in the spreadsheet.
  6. Click Save.
