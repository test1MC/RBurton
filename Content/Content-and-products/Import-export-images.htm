Importing and export images

This import allows you to assign additional assignments to products as well as product options.

When updating by spreadsheet, you should always export first to ensure you are using the most recent spreadsheet version. This will also guide you on the proper format for each column and row as well as which fields are required.

Note that the fields of the exported image spreadsheet will differ depending on whether you are exporting an image spreadsheet for products or an image spreadsheet for products options.

To export an image spreadsheet
  1. In Content, select Import/Export > Images.
  2. In the Export Images area of the screen, select either Products or Products Options from the list menu.

  3. Click the Export to Excel button .

    A link to the spreadsheet appears in the Current Exports list.

  4. Click the spreadsheet link to download and open the file containing image information. Make modifications as needed, and save it to your computer. Fields contained in the spreadsheet are:

    Field Description
    Products
    product_id Tthe ID automatically assigned to the product in the Aptos database when the product was created. Used for lookup purposes and cannot be modified at any time.
    DisplayName The name of the product as displayed on your site. Used for lookup purposes and cannot be modified via the image import.
    PartNumber The manufacturer’s part #, internal stock #, style # or any other uniquely identifying # for this product. Used for lookup purposes and cannot be modified via the image import. If you are using SKUs to manage your internal stock, this is the number that will be used to tie the option-combination to the product. This is the same as the value entered into the Value (SKU) field of the Content > Products > Options page.
    ImageName The name of the image file that has been uploaded to the Item Images folder on the FTP server. Be sure to include the image type extension, for example: image.jpg.
    ImageCaption Depending on how your site is wired, this value may be used to populate the image alt text.
    ImageOrder Controls the order in which the alternate images are displayed. Images are displayed from lowest to highest numbers (that is, 0, 1, 2, 3…).

To import images

This import allows you to assign additional images to products or product options. It also allows to you configure the caption and display order of those images.

  1. In Content, select Import/Export > Images.
  1. In the Import Images area of the screen, select whether you're uploading Products or Products Options images from the list menu.
  2. Click Browse and navigate to the Excel file containing the image data you want to upload.

  3. Click the Import button to upload the file information into the site manager.
