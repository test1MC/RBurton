Using the Job Scheduler

Use the Job Scheduler to view the recurring jobs configured for your site (such as imports and exports) and modify the schedule for those jobs. You can use this service to update job schedules and frequencies without the need for Aptos support.

Custom jobs are scheduled at Content > Tasks > Job Scheduler.

Not all recurring jobs use the Job Scheduler service. Engage with your Project Manager, Account Manager or Client Care to implement any jobs with the job scheduler.
Managing scheduled jobs

Scheduled jobs are listed in the window's Manage Existing Scheduled Jobs section. The schedule settings for each job are controlled in this area. Click More Details ... next to the job to see most of these options and settings.

Job scheduler interface
Element Description
Job ID The ID automatically assigned in the database when a scheduled job is created.
Job Name The name of the job.
Active Whether the job is enabled to run.
Description What the job does.
Triggered by Select the "schedule" trigger from the drop-down menu. See Scheduling jobs below for more information.
Email on Error If enabled, sends email when an error occurs. Emails are set up by Aptos only in the detailed Job Steps and can be configured to go to clients, Aptos staff and Aptos technical support.
Abort on Schedule Miss Whether to abort the job if the schedule was missed. Take for example, a job scheduled to run every hour on the hour, and so runs at 12 PM. If the job is still running at 1 PM, it will not run again at 1 PM. Instead, the job will try to run at 2 PM, as long as the previous job is complete.
Abort on Error Select to abort the job in case of an error.
Job Scheduler Environment Jobs are often configured in the Job Scheduler's staging database before they are set to run in the production database. To view the jobs configured in one database instead of the other, select the Staging or Production radio button.
Copy Job Click to copy an existing job. This is helpful if you are making minor changes when scheduling a new job.
Move Job to Different Environment Click to move a job from the Job Scheduler's staging database to the Job Scheduler's production database.
View Job History View the Job History by step, when it last ran, the date it was created, and the results. You can filter by date range and job step.
View Detailed Logs

View the Detailed Logs by job step and by log file.

When the job is set up, click to Save the new job.

 Scheduling jobs

Click on the Edit button to schedule a job; the Job Schedule pop-up window opens.

The scheduling settings are:

Schedule Type:
  • One Time Only—Choose this option, and select a date and time, to run the job only once; select a date and time.
  • Recurring—Choose Recurring from the drop-down menu to configure these additional settings for recurring jobs in the pop-up window.
 Recurring Occurrence

Recurs: Choose whether you want to schedule the job to run daily, weekly, or monthly.

  • Daily - Select the day(s) of the week you want the job to run.
  • Weekly - Enter the number of weeks and choose the day(s) in which you want the job to run.
  • Monthly - There are two options with this choice. You can select the date(s) on which you want to schedule the job. Or, you can choose the radio button and select the first, second, third, fourth, or last (day of your choice) of the month.
 Times

There are two options for choosing daily frequency:

  • Occurs once per day: Enter the exact time you want the job to run; click the alarm clock icon to choose a time.
  • Occurs every: Sets the daily frequency to occur every X minutes depending upon the number entered in the field. In addition, set a starting at time and an ending at time.
 Duration

Set the Start Date and an End Date. You can also choose to set No end date.

Save the scheduled settings and close the pop-up window.

 Creating a job from a template

Select the site you want to work with from the drop-down menu—these may be Development, Staging, Production, or a Site Manager Task—then proceed.

  1. Select the Create Job from Template button located at the bottom of the page. A pop-up window opens.
  2. From the form located in the pop-up window, select the template from which you want to create the job and then Save. This creates an inactive job based on the template selected.
  3. Close the pop-up window and the newly created job is displayed on the page. Modify and schedule the job as necessary and activate it when needed.
