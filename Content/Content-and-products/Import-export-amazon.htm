Importing Amazon inventory

Aptos has created an import spreadsheet that allows clients to enter ASINAmazon Standard Identification Number; a 10-character alphanumeric unique identifier assigned by Amazon.com and its partners for product identification within the Amazon organization. or SKU numbers. When the spreadsheet is imported, accesses the Amazon database, and pulls in product information. This allows a more efficient product import process for clients who have access to either generic Amazon ASIN numbers, or ASIN numbers that they use while selling on Amazon.

It is important to note that the ASIN Import is a one way communication. Aptos imports information from Amazon. To export data to Amazon, you must enable the Amazon Channel. Talk to your Client Manager or Technical Project Manager to learn more about how to sell your products on Amazon.

Only the first 250 ASIN/EISBN/ISBN/EANs will be processed due to Amazon constraints. The Product Advertising API supports ItemLookup batch requests only when the ItemType is an EANEuropean Article Number , ISBNInternational Standard Book Number, EISBNElectronic International Standard Book Number, or ASINAmazon Standard Item Number.

To import Amazon inventory from an Excel spreadsheet

  1. In Content, select Import/Export > Amazon Inventory.
  2. Click Browse to select the local source where your import Excel spreadsheet resides.
  3. Select whether to Import products as inactive.
  4. Then click the Import button .
Clients are responsible for reviewing all content, including pricing and options, after import is complete.

About Question and Answers spreadsheet fields

This is the information mapped from Amazon to the Aptos Digital Commerce platform upon import:

Information Mapping
Primary Product Image pulls the primary product image, and associates it with the Main Product Image.
Amazon SKU Maps to Part Number.
Amazon Product Group Maps to Sub Category.
Amazon Manufacturer Maps to Brand.
Amazon Brand Maps to Brand if Manufacturer is empty.
Amazon Width, Length, and Height Added to the Product Details Width, Length, and Height.
Amazon Weight Mapped to the Shipping Weight .
Amazon Merchant item attributes description Maps to Short Description.
Amazon Title Maps to Display Name; strips the brand off of the Amazon Title.
Amazon Low Price Maps to Sells For price.
Amazon Features Features map to the field Xtra5 in the product spreadsheet.
Amazon Tags Map to Tags, and are entered comma delimited.
For Parent ASIN IDs, Aptos also pulls in the options associated with the Child products.
  • For Product Option prices/mark-ups in the Site Manager, deducts the Child product price from the Parent product price, and uses this value as the Option mark-up.
  • Amazon Part Number if no Part Number, then Amazon Display Name maps to the option Group Name and Group Alias.
