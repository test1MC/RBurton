Importing and exporting pricing tiers

Use this feature to import and export tiered prices using an Excel spreadsheet.

You can export a spreadsheet with all current price tiers, or to a blank spreadsheet to use as a template when importing.

When updating by spreadsheet, you should always export first to ensure you are using the most recent spreadsheet version. This will also guide you on the proper format for each column and row as well as which fields are required.

Be sure your browser is set up to allow pop-ups from the site manager.
To export pricing tiers
  1. In Content, select Import/Export > Price Tiers.
  2. Choose to export either current tiers, or a blank template:
    1. To export a spreadsheet with your current tiers, click the Export Current Tiers to Excel button. This generates an Excel spreadsheet, and a link to it. Click this link to open, modify, and or save it to your hard drive. See Price tiers spreadsheet fields for field descriptions.
    2. To export a spreadsheet you can use as a template, click the Export Template File To Build Your Pricing Tiers button. This generates a sample record for the first 10,000 products in under a simulated tier called ‘Sample’ and a tier price=the actual price.

To import pricing tiers
  1. In Content, select Import/Export > Price Tiers.
  1. Click Browse, find the Excel spreadsheet you want to import, and click Open.
  2. Click the Import button .

    Pricing tiers are automatically uploaded to the server, and each new price tier is assigned a price ID used to identify it in the database.

Import notes

  • Each Product_Id can be listed more than once—once per each tier of price the product has. If Product_Id is invalid, then ignores it and displays an error message at the end of processing.
  • If the Tier is invalid, then ignores it, and displays an error message at the end of processing.
  • Importing does not allow for adding Tiered Groups from this screen; you must pre-add the Tier Groups before importing the spreadsheet.
