Importing and exporting price breaks

You can enter a single price break either directly in the site manager, or through the Product Import/Export spreadsheet. However, if you have multiple Price Break tiers, you need to use the Price Break spreadsheet.

Price Breaks can also be entered directly into at the Product Level Details.

When updating by spreadsheet, you should always export first to ensure you are using the most recent spreadsheet version. This will also guide you on the proper format for each column and row as well as which fields are required.

To export price breaks
  1. In Content, select Import/Export > Price Breaks.
  2. Choose whether to Export Products price breaks by Sub-Categories or Brands using the list.
  3. Select the brands or categories for which you want to export price breaks. Select more than one by holding down the Ctrl key. You can also use the Select All and Clear All links.
  4. Select whether to Export in Pipe Delimited format.

  5. Click the Export to Excel button .

    This creates an Excel spreadsheet and a link to it at the bottom of the page. Click this link to open, modify, and or save it. Be sure your browser is setup to allow pop-ups from the site manager. Price break spreadsheet fields you can modify are:

    Field Description
    product_id The system generated ID number of the current product.. When present, this ID # tells that this is an update to an existing record already present. If it is blank, it will create an entirely new record.
    DisplayName The front end pre-defined display name of the product.
    PartNumber The pre-defined manufacturer’s part #, Internal Stock #, style # or any other uniquely identifying # for this product.
    BreakQuantity The quantity threshold a customer must reach to achieve the discounted rate.
    BreakLevel

    Price breaks may be set up at the sub-category, product, and product option level. Because this spreadsheet can update price breaks on each level, you must designate at which level the break applies in this field. Enter:

    • "S" for a sub-category,
    • "P" for a product,
    • "O" for option level.
    BreakPrice The new price for the product that the customer will pay when in this tier.
    BreakType

    The type of price break can be set as a Set Price, Fixed Adjustment, or Percentage.

    • For a Set Price, the value entered replaces the Sell For price of the product.
    • The Fixed Adjustment type allows you to enter a value that will either markup or markdown the Sell For price.
    • A percentage value offers a percentage discount off of the product's Sell For price.
    GroupName Price breaks may be made available to specific tier groups (optional). Enter in the name of the tier group for the tier group you wish to make a price break available (can be overridden by the "Apply to All Shoppers" check box in Products > Add/Edit > Details > Pricing > Price Break Details).
    Delete True or False field. Enter True in this column if you want to delete the price break.
To import price breaks
  1. In Content, select Import/Export > Price Breaks.
  1. Click Browse, find the Excel spreadsheet you want to import, and click Open.
  2. Click the Import button .

    Price breaks are automatically uploaded to the server.
