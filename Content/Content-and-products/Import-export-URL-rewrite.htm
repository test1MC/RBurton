Importing and exporting URL re-writes

You can customize the URL structure on some content pages dynamically. Changes made on this page will overwrite the standard platform URL structure. You can use spreadsheets to import and export URL re-writes. See Maintaining URL re-writes for more information.

When updating by spreadsheet, you should always export first to ensure you are using the most recent spreadsheet version. This will also guide you on the proper format for each column and row as well as which fields are required.
To export URL re-writes
  1. In Content, select Import/Export > URL Rewrite.
  2. Use the File Type list to select the kind of file you want to export.
  3. If needed, click the Import/Export Page Url Rewrites banner to expand that area of the screen.
  4. Click the Export to Excel button . This creates an Excel spreadsheet containing URL re-writes of the type you selected, and a link to it in the Report area below.
  5. Click the spreadsheet link, add or update category information in it, and save the spreadsheet to a local drive.
 To import URL re-writes
  1. In Content, select Import/Export > URL Rewrite.
  2. Use the File Type list to select the kind of file you want to export.
  3. If needed, click the Import/Export Page Url Rewrites banner to expand that area of the screen.
  4. Click Browse, find the Excel spreadsheet you want to import, and click Open.
  5. Click the Import to Excel button .

    URL re-writes are automatically uploaded to the server.
