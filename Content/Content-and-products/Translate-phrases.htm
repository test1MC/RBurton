Translate phrases

Use this page to translate your site into different languages. After setting up all of the languages you want to use, set up all of the phrases appearing on your site you would like to translate. This is for translated phrases that appear on both the front end and back end (backadmin) of your site.

It can be more efficient to break phrases into multiple parts, or even separate words, if a phrase segment or word appears in several places in the site.
To add phrases
  1. In Content, select International > Translate Phrases.
  2. In the Original field, type the phrase you want to translate.
  3. The Location Optional list menu presents the preset values for where the phrase should be displayed. Select a value and click the plus sign (+) to add the type value to the list below. This value is saved in EOM.

  4. Click the + Add Phrase button.

    The phrase and any additional values are added to the translation table, and displayed in alphabetical order on the lower half of the page.

  5. On the list of assigned types, you can remove the value by clicking the minus sign.
 Importing and exporting phrases

You can import and export phrases and equivalents using an Excel spreadsheet. The five most recent exports are also available in this area.

When updating by spreadsheet, you should always export first to ensure you are using the most recent spreadsheet version. This also ensures that the columns and formatting are entered correctly in .

To export phrases
  1. In Content, select International > Translate Phrases.
  2. Select the language of phrases you want to export.
  3. Click the Export button. This creates an Excel spreadsheet containing all phrases for that language, and a link to it below the Import / Export buttons.
  4. Click the spreadsheet link, add or update rule information in it, and save the spreadsheet to a local drive.
 To import phrases
  1. In Content, select International > Translate Phrases.
  2. Click Browse, find the Excel spreadsheet you want to import, and click Open.

  3. Click the Import button.

    Phrases are automatically uploaded to the server.

 To search for phrases
  1. Click the black Search banner to expand that area of the screen if necessary.
  2. You can search for all phrases, or to narrow your search for existing phrases, use these three fields.
    1. Keyword - searches the “original” fields and returns results in alphabetical order. Translated content is not searched.
    2. Location - Searches by location type; can be selected for search without keyword content, or other selectors.
    3. Language - Searches by language; may be selected for search without any keyword content, or other selectors.
  3. To return results for phrases that have no content in the translation, select Only search empty translations. A Language must be selected to search for an empty translation.

  4. Click Search.

    Phrases are listed in the Translations area below search criteria.

 To edit phrases

  1. Phrases displayed in the Translation area are listed by default in alphabetical order. Click on the Type, Original, ID or Language header to sort the results. Click again on any header to sort in descending/ascending order.

  2. Click the Edit icon of the phrase you want to edit.

  3. To add a Location Type, select it from the list menu, and click the Add icon .
  4. To delete a Location Type click its Delete icon .
  5. Change or add Original and Translation text as needed.
  6. Click Save.
