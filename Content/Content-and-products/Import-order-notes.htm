Importing order notes

Each time an order status changes, notes are added to the Order History. Notes can also be manually added as needed. You can view order notes recorded between specific dates and by the user who made a change to the orders. Information displayed includes:

  • order # – This number identifies the order that the note applies to.
  • Note Date – The date and time the note was generated or entered.
  • Sales Person– The user who made the change to the order.
  • note – The record change to the order which appears in the Order History section of the Order Details window.

To view order notes

  1. In Content, select Import/Export > Order Notes.

  2. Select the desired date ranges from the drop down menus and select the individual user that you want to pull records from in the Note Written By list.

    If you don't select a user, all note records are pulled for the specified date range.

  3. Click the Search button.

To import order notes

  1. In Content, select Import/Export > Order Notes.

  2. Click Import Order Notes to expand that area of the screen.

  3. Click Browse, find the Excel spreadsheet you want to import, and click Open.
  4. Click the Import button .
