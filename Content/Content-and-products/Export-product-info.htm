Exporting product information

You can export and import products to a spreadsheet for mass updates of existing products or bulk uploads of new products. Nearly every attribute of a product can be manipulated using this spreadsheet.

has two distinct types of products: Products with Options and Products without options. All products can be exported by using Product Export, but the export will contain only the information that is pertinent to the Parent level product. All option-level information must be updated using Import/Export > Product With Options

When updating by spreadsheet, you should always export first to ensure you are using the most recent spreadsheet version. This will also guide you on the proper format for each column and row as well as which fields are required.

Be sure your browser is setup to allow pop-ups from the site manager.

To export product information

  1. In Content, select Import/Export > Product Export.
  2. Choose whether to Export Products By either individual Sub-Categories or Brands using the list.
  3. Select the brands or categories for which you want to export product information. Select more than one by holding down the Ctrl key. You can also use the Select All and Clear All links.
  4. Use the check boxes labeled Include to select what to include in the file:

    1. products in shared sub-categories

    2. deleted products

    3. inactive products

    4. inventory only

  5. Click the Export to Excel button .

    A pop-up window opens showing details of the scheduled export.

  6. Make any changes needed to the export schedule date and time, and notification email address if needed, and click Save.

    After the export has run and completed, a link to it appears at the bottom of the main screen.

    You may need to refresh your browser window for the link to appear.

  7. creates an Excel spreadsheet and a link to it at the bottom of the page. Click this link to open, modify, and or save it. Product information spreadsheet fields you can modify are:

    Site_ID Default value is 1. Will be different if more than one site is sharing pricing/product information. Required.

    The name of the actual image to use on the FTP server.

Once this file is processed, the jpg is copied to the new name automatically. If the same is used for multiple products, that image will be pulled for each of them, so only one image would be required.
