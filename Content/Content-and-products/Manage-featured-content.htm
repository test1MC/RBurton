Managing featured content

Use Featured Content to update the front end of the site dynamically. There is no need to submit a request or contact your Client Manager. During the design phase of the site implementation, certain areas of the site can be identified as dynamic content areas and can be “wired” to pull their content from specific Featured Content areas.

Currently, items in the Content Tab that allow for featured content are:

  • Categories,
  • Sub-Categories,
  • Brands
  • Pages.
If your site has not been wired to accommodate featured content areas during the implementation phase, and you would like to be able to easily edit certain areas in your site, contact your Client Services Manager; they will scope it out for you.

To configure featured content

  1. In Content, select Categories > Featured Content.
  2. Select the Category you want to configure from the list.
  3. If applicable, select an appropriate Tier Group from the list.
  4. Click the Add button (+).

  5. In the pop-up window that appears, enter a name for the Featured Content Block Title for wiring purposes, and so you’ll be able to relocate the area when you need to update it.

    For example: Social Media images/links

  6. To add HTML content:

    1. For Content Type, select the HTML radio button.

    2. Within this window you can use either the Editor View tab to use the WYSIWYGwhat you see is what you get editor to add the content, or you can click on the HTML View tab and enter in actual HTML code for the content.
    3. Click Save.

  7. To add image content:

    1. For Content Type, select the Image radio button.

    2. Browse for the image file that you want to upload from your computer.

    3. In the Link URL field, enter the http://URL for the image. This is the specific page the customer is taken to either within or outside of your site when they click the image.
    4. In Image Alt Text, enter the text that appears on the screen when the customer hovers over the image.
    5. Choose a Schedule from the list if appropriate.
    6. Enter Description text for the image.
    7. Click Save.

Scheduling featured content

If your site is configured to display featured content, you can configure different versions of that content, each with its own publish date and time. If, for example, you want new content to display starting in the middle of the night or while you are away from your computer, you can configure and save the new content at a more convenient time and simply assign the date and time that the content should be published.

This feature applies to featured content under the Pages menu as well as featured content by Category, Sub-Category, and Brand.

To add date/time-specific content to a featured content block

  1. Navigate to the block in question, for example Content > Pages> Featured Content.

  2. Click the Edit icon to display the Add/Edit Featured Content pop-up window.

  3. From the Schedule list, select Add Content .

    Note that the Type radio buttons and Title text box are now disabled

  4. Enter updated content in the WYSIWYG editor.
  5. In the Publish On field, select the date and time for the content to be published.
  6. Click Save.
