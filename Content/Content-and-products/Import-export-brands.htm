Importing and exporting brands

You can enter brands in bulk using a spreadsheet.

Before exporting brands, be sure your browser is setup to allow pop-ups from the site manager.

You should always do an export to ensure you are using the most recent spreadsheet version. This will also guide you on the proper format for each column and row, as well as which fields are required.

To export brands to an Excel spreadsheet

Before exporting brands, be sure your browser is setup to allow pop-ups from the site manager.
  1. In Content, select Import/Export > Brands.

  2. Click the Export to Excel button .

  3. Click the file's link in the Report column to view it, or to save it to your hard drive.
  4. Modify contents of the spreadsheet as needed, save your changes and import when done. Fields that can be updated are:
Field Description
Brand_ID System Generated. Leave blank if you are entering a new brand. If you are updating a brand, enter the pre-defined system generated Brand_Id.
Brand The name of the brand or manufacturer. Required.
Brand_Desc The text for the brand description. Normally shows up on the Brand page for each brand.
Brand_Cat_Url The brand's or manufacturer's home page if you want customers to be able to view their website from your brand page.
Title The meta title for the brand.
Keywords Keywords associated with the brand.
Description The meta description of the brand.
CSS Enter the CSScascading style sheet file to be used for the brand page. If you are not familiar with CSS, leave this field blank.
Tags Any tag words associated with the brand. These improve inside search and SEOsearch engine optimization for the brand.
Brand_Header_Image If the site is wired to use this, enter the image to be displayed on the brand page.
UPC The UPC associated with the brand or manufacturer.
Brand_Image The image path for the brand logo.

To import brands from an Excel spreadsheet

  1. In Content, select Import/Export > Brands.
  2. Click Browse , navigate to the spreadsheet you want to import, and click Open.

  3. Click the Import button .

    The brands are automatically uploaded to the server. Each new brand is assigned a brand ID used to identify it in the database.
