Importing and exporting categories

The Content > Import/Export > Categories page provides the same functionality as the Content > Categories > Add/Edit page. Here you can add or edit categories individually, or import an Excel spreadsheet to create or update multiple categories at a time.

When updating by spreadsheet, you should always export first to ensure you are using the most recent spreadsheet version. This will also guide you on the proper format for each column and row as well as which fields are required.
To manually add or edit categories
  1. In Content, select Import/Export > Categories.

  2. At the bottom of the page, click Add (+).

    This adds a new row to the top of the list with editable fields you can use to define category information. Each new row is automatically assigned a List ID number uses to identify a specific category.

  3. Add information which defines the category as appropriate; see the New category field and element descriptions table for details.

New category field and element descriptions
Field or element Description
Delete Click to delete this category.
List ID Automatically generated when creating a category; used by to identify the specific category. Do not modify when making edits to a category.
Category Name Enter the name of the new category.
Translate Click the speech bubble to configure the name of your category to display in different languages.
Order You can assign the order in which you want your categories to display depending on how your site is wired.
Hidden

Whether this category is displayed or not; checked is hidden. The checkbox's true/false value is written to the new Hidden database field for the selected Categories.

Depending on the settings of your site, this feature may require additional wiring. Contact your Client Manager for more information.
Display on Mobile Used to limit which categories are made available to mobile device users.
Image If your category is wired to associate an image with it, click the icon in the Image column and the Image modal window will open. Click the Browse button to select the appropriate image from a local source. Click the Save button to save the selected image. An image will now be associated with the category.
SC Image Used in place of the main image if your site is wired to display products on a social commerce site.
Meta & Heading Use to enter your meta title, meta keywords, meta description and heading in the corresponding fields. More information about meta content and Category Headings can be found in Google’s SEO guide.
Sub-Categories

Click the folder icon to assign previously-created sub-categories to their corresponding categories.

Select the related sub-categories. When a sub-category has been assigned to a category, the check box becomes grayed out. As before, remember to Save when you have completed making your selections.

To export categories

  1. In Content, select Import/Export > Categories.

  2. Click the black Import/Export Categories bar.

    The page expands to show the Import/Export Categories area.

  3. Click the Export to Excel button .

    This creates an Excel spreadsheet and a link to it beneath the Import/Export buttons.

  4. Click the spreadsheet link, add or update category information in it, and save the spreadsheet to a local drive.

 To import categories
  1. In Content, select Import/Export > Categories.

  2. Click the Import/Export Categories menu bar.

    The page expands to show the Import/Export Categories area.

  3. Click the Excel File field's Browse button.
  4. Browse to and select the Excel file you want to import, and click Open.
  5. When the name of the file you chose appears in the Excel file address field, click the Import button to upload new and changed category information in the spreadsheet.
  6. Click Save.
