Importing and exporting promotions

Use this spreadsheet to import and export all promotion codes on your site. You can export all deleted promo codes by themselves or include all deleted promo codes when exporting a file.

When updating by spreadsheet, you should always export first to ensure you are using the most recent spreadsheet version. This will also guide you on the proper format for each column and row as well as which fields are required.
To export promotions
  1. In Content, select Import/Export > Promotions.
  2. Use the check boxes to select whether to:
    1. Export ROSrun on station hidden promos only
    2. Include deleted promos.

  3. Click the Export to Excel button .

    This creates an Excel spreadsheet and a link to it in a green success banner at the top of the window.

  4. Click the spreadsheet link, add or update category information in it, and save the spreadsheet to a local drive.

    Possible values and explanations for spreadsheet columns are:

    Promotions spreadsheet fields
    Spreadsheet column Possible values and meanings
    CouponType

    The Type of Promotion you are running. This list can be found when adding a new promotion in the site manager.Required.
    CouponName

    A unique name for the promotion. If this is an update to an existing promotion, it updates the existing record.Required.
    Deleted

    True or False; indicates whether the promotion is still active. Required.
    CouponDesc

    XML for the existing promotion. To view the XML format, by edit a promotion of this type in the site manager. Required.
    AutoTrigger

    True or False; indicates whether this promotion is auto-triggered in the shopping cart. Required.
    beginDate

    The starting date of the promotion. The time can be included for a flash sale starting at a certain time. Format is MM/DD/YYYY HH:MM AM/PM. Required.
    HideInPromoLists

    True or False; if f true, this promo will not show in the promotion code lists. If you are importing a large promotion code list and the call center will not be using these, set to true to avoid performance issues. Required.
    Affiliate

    Comma separated list of affiliate codes if restricting by affiliate, otherwise leave this blank.
    excludeAffiliate

    True or False; required only if affiliate is specified. If blank, it is treated as false. If true, this applies the promotion to all affiliates EXCEPT those in the affiliate list.
    Default_Coupon

    True or False; only one promotion code can be set to true at a time. If true, the Order Review email uses this promotion code as the baseline for generating unique one-time use codes.
    oneTimeCoupon

    True or False; indicates whether this is a one time use promotion code. If blank, it defaults to false.
    ignoreAffiliateforAutoTrigger

    True or False; override affiliate constraint if typed in during shopping cart. If blank, it defaults to false.
    publicCoupon

    True or False; indicates whether this promotion code will show up as a public promotion if an affiliate logs in to the affiliate section. If blank, it defaults to false.
    CouponShortDesc

    Short description for the promotion. For BOGObuy one - get one promotions, this is the description that appears in the shopping cart.
    CouponImage

    Directory path to an image for the promotion code. If Aptos has wired your site to show a custom promotions page, this is the image that appears. It is a relative path from the root of the site, for example, images/promos/image1.png.
    CouponLink

    A link to a page containing more information about this promotion code. If the promotion code is displayed on a page, and a deep link to another page is needed, specify it here. This is relative from the root of the site as well.
    RepeatCustomer

    Trueor False; if true, this Promotion Code is based on previous purchases and NOT the current cart—the user must be logged in. If blank, it defaults to false.
    RepeatTimePeriod

    If RepeatCustomer is true,the number of months from purchase this promotion is valid. If blank, defaults to 12
    OnePerCustomer

    True or False; indicates whether this promotion code is only able to be used one time per customer. If blank, it defaults to false.
    FreeShippingAddOn

    True or False; when true, free shipping is added on to the existing promotion. If blank, it defaults to false.
    FreeShippingMinCartAmount

    The minimum dollar amount in the cart needed before free shipping is applied to the promotion. If blank, defaults to 0.
    tierGroups

    Comma separated list of customer groups if promotion is to be restricted by customer group, otherwise leave blank.
    excludeTierGroups

    True or False; required only if affiliate is specified. If it is blank, it is treated as false. If true, this applies the promotion to all customer groups EXCEPT those in the customer group list. If blank, defaults to false.
    includeTierGroups

    True or False; required only if affiliate is specified. If it is blank, it is treated as false. If true, this applies the promotion to all customer groups in the customer group list. If blank, defaults to false.
    flashAmount

    An Integer representing the number of times this promotion can be used before it is disabled. If 0, it will never auto-disable
    triggerOnBirthday

    True or False; if true, this coupon is valid only on the customer's birthday. If blank, defaults to false.
    ApplyToHighestPricedItemOnly True or False; if true, applies promotion only to highest priced cart line item total (quantity * price). If blank, defaults to false.
    ApplytoHighestCartLineOnly

    True or False; if true, applies promotion only to highest priced cart line item (price only). If blank, defaults to false.
    PromoGroup

    A unique name for the promo group associated with this Promotion. If the promo group doesn't currently exist, it will be created with no limitations.
To import promotions
  1. In Content, select Import/Export > Promotions.
  1. Click Browse, find the Excel spreadsheet you want to import, and click Open.
  2. Click the Import button .

    Promotions are automatically uploaded to the server.
