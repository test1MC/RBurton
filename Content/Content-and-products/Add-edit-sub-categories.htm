Adding and editing sub-categories

Use the Add/Edit page to create new sub-categories, assign them to a pre-defined category, and edit various aspects of your sub-categories including meta content, page headings, and more.

Note: A sub-category name can only include letters and numbers. If a sub-category has yet to be assigned to a category, you can locate it under No Category Assigned.

To add a new sub-category
  1. In Content, select Sub-Categories > Add/Edit.
  2. Enter a new sub-category Name.
  3. Enter Meta-Title information.
  4. Select a Category in which the sub-category is to appear.

  5. Click Add (+).

    The page reloads with the Category selected in the drop-down in the Existing Sub-Category section, and its sub-categories displayed below, including the new one you just created.

  6. Scroll to the bottom of the screen if necessary, and if you want to Create 301 redirects for this sub-category you are editing, enable it here. To learn more about 301 redirects, see knowledge base article #1068 for more information.
  7. Click Save.
 To edit an existing sub-category
  1. In Content, select Sub-Categories > Add/Edit.
  2. Find the sub-category you want to edit:
    1. To limit search results to within a specific category, use the list to select the Category. This is helpful if you have a large taxonomy.

    2. You can enter part of the sub-category name, id, or alias in the Sub-Category field and click Search to narrow search results even more.

  3. Sub-category editing options are

    1. Delete ID—deletes a particular sub-category.

      Before you delete a sub-category, consult with your Client Manager. Deleting a sub-category may unassign all products that exist within it. You may wish to mark a sub-category as hidden as an interim solution.

    2. Where Do I Live? button —Use to trace backwards through the Multi-Level Tree path to determine the category to which a sub-category is assigned. A pop-up window displays the taxonomy of the sub-category in question.

    3. Preview button —Click to view how the sub-category appears on your site.

    4. Display Order—Used to dictate the order in which sub-categories are displayed. Number sub-categories sequentially, with sub-category '1' appearing first.

      Sub-categories with a Display Order of '0' are displayed alphabetically.

    5. Hidden—Use as an alternative to deleting the sub-category. It hides the sub-category so customers won't be able to search for or see it. Uncheck this box when you want the sub-category to appear.

    6. Advanced—Click to allow the selected sub-categories to be visible for the affiliates chosen from the list. In addition, click the Create 301 redirects check box next to the Save button at the bottom of the screen.

    7. Show in Search—Allows the sub-category to be searchable. If a sub-category Hidden, you may want it to also be hidden from search.
    8. More Details—Click to display editable fields and options for this sub-category. See the More Details tables for descriptions.
  4. Click that sub-category's Save save buitton.

Meta Description

Describes the content within the sub-category, giving search engines' spiders an accurate summary filled with multiple keywords. The Meta description is especially important because it's the only tag supported by some engines. You may wish to use marketing copy in the meta description field for optimal phrasing.

Meta tags are hidden in a document's source and are invisible to the reader. Some search engines, however, are able to incorporate the content of meta tags into their algorithms. No engines penalize sites that use meta tags properly, so it's recommended that you always include them.

Price Breaks

This function allows you to create price breaks at the sub-category level, in addition to the individual product level. In other words, customers can be given discounts when they purchase multiple products from the same sub-category.

To manage Price Breaks

  1. Click Add/Edit located under Price Breaks. Make changes as needed in the pop-up window:
    1. Quantity Range: Enter the starting and ending quantity for the price break.
    2. Value: Enter the price adjustment, either a dollar amount or a percentage depending on the Type selected.
    3. Type: Select Set Price, Fixed Adjustment, or Percentage from this list:
      1. Set Price: This value will override the regular price. For example, if a customer buys 6 or more items in this sub-category, they would be $15 each if the value is set to 15.
      2. Fixed Adjustment: This value is deducted from the regular price. For example, if a customer buys 6 or more items in this sub-category, if the value is set to 5, they could save $5 on each item.
      3. Percentage: This value is deducted as a percentage off the regular price. For example, if a customer buys 6 or more items, they save 10% on each item if the value is set to 10.
    4. Tier Group (Optional): If desired, select a tier group to which this price break should apply.
    5. Display Price Adjustment as Separate Item with Label: If you choose, you can select the check box and enter a value to show the price adjustment as a separate item in the shopping cart with the label you create here.
  2. At the end of the adjustment, click Add (+) to add this price break.

Maximum Cart Qty

Used to restrict the maximum quantity of a product in a customer's cart for a whole set of products, for example, samples of a product. After setting the maximum quantity, you can check by:

  1. Go to the front-end of the site.

  2. Select a product from the sub-category.

    The Quantity in the product page should only allow for as many as the number you have entered at the sub-category level.

  3. Add the product to the cart. The maximum quantity should match the number you entered at the sub-category level.

    The quantity drop-down menu on the cart page will populate the Min Cart Qty (minimum) value using the same logic used to populate the Max. Cart Qty (maximum) value as follows:

    1. ​ If you enable the Allow Backorder setting at Users > Website Settings > Site Layout > Shopping Cart Settings, and if Max Cart Qty is set greater than 0, the maximum value in the cart drop-down menu is equal to the sub-category maximum value. Both the product quantity and option quantity are ignored.
    2. If you disable Allow Backorder at Users > Website Settings > Site Layout > Shopping Cart Settings, and the Sub-Category Max Cart Qty is not set here, then the maximum value in the cart list menu is by default 250.
    3. The minimum cart quantity specifies the minimum the customer can choose.
    4. The increment determines the next quantity allowed in the cart. For example, if a minimum of 10 is specified, and increments of 10 need to be purchased with a maximum of 50, the cart will show 10, 20, 30, 40 and 50 as possible option quantities.
Qty Increments Used to set the Incremental quantity of a product in a customer's cart for a whole set of products, for example, increment of two quantities per product.
Channel Low Threshold

Used to manage your inventory across channel marketplaces. It helps you set a "channel low threshold" at several levels in order for a product to be present in the inventory sync with multiple channel marketplaces.

You can set the low threshold at the product level, the sub-category level, or at the channel marketplace level.
