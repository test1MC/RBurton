Adding and editing classifications

Use this add and edit classifications. It also contains your existing classifications including: Amazon, Overstock, Rakuten, PriceGrabber, and any others that you have set up.

 

About classification groups

The top level classifications are displayed in the Classification Groups panel. There are specific settings that must not be changed.

•Default - This setting is in place within the Site Manager. This the only Classification Group that displays on the front end of your website. It will display as filters on your site, if it is used. To ensure proper functionality, the name "Default" must not be changed.
