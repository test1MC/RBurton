Importing order cost

Use import order cost to import a revised cost directly into an order. This is useful if you need to make modifications to a customer order.

The feature accepts an Excel spreadsheet with three columns of data:

  • OrderNumber – The order number representing the order to apply a revised cost.

  • Cost – The cost for the related orderNumber and SKU.

  • SKU – If specified, the cost supplied is applied to the item matching the given SKU. If SKU is left blank, the cost is distributed among each item equally in the order.

To import order cost

  1. In Content, select Import/Export > Order Cost.
  2. Click Browse, find the Excel file you want to import, and click Open.
  3. Click the Import button .
