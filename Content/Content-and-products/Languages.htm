Setting up languages displayed on your site

You can translate your site into different languages. The first step in setting up site translation is to add or edit the language you want to integrate. This is for translations that appear on both the front end and back end (backadmin) of your site.

To add a language to display on your site

  1. In Content, select International > Languages.
  2. Choose a Language Name from the list.
  3. Click the Add Language (+) button to save the new language.

To edit an existing language

  1. In Content, select International > Languages.
  2. Find the language in the grid displayed;.
  3. Modify the appropriate values as needed, for example Display Name or ISO Code.
  4. Click Save at the bottom of the page.
