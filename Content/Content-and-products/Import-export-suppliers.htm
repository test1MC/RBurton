Importing and exporting suppliers

You can enter suppliers in bulk using a spreadsheet.

You should always do an export to ensure you are using the most recent spreadsheet version. This will also guide you on the proper format for each column and row, as well as which fields are required.
To export suppliers to an Excel spreadsheet
Before exporting suppliers, be sure your browser is setup to allow pop-ups from the site manager.
  1. In Content, select Import/Export > Suppliers.

  2. Click the Export button.

  3. Click the file's link in the Report column to view it, or to save it to your hard drive.

  4. Modify contents of the spreadsheet as needed, save your changes and import when done. Fields are:

To import suppliers from an Excel spreadsheet
  1. In Content, select Import/Export > Suppliers.
  2. Click Choose File, navigate to the spreadsheet you want to import, and click Open.

  3. Click the Import button.

    The suppliers are automatically uploaded to the server. Each new supplier is assigned a Supplier ID used to identify it in the database.
