Adding and editing product options and option groups

Product options are the variants for each product that allow you to offer different choices to your customers such as: size, color, style, and so on. For example, if you sell shirts, the same shirt might be available in sizes small, medium, large and extra large. The shirt might also be available in solid colors or plaids.

Product option groups are configured at Content > Products > Options. To create an Option Group and associate options to that group, follow the steps in this topic. You can also create multiple product option groups with the import/export product options spreadsheet. Click here to learn more about how to use this feature.

After you have set up the option groups , you will still need to apply the options at the product level in Content > Products > Add/Edit > Details > Options.
Adding an Option Group

To create an Option Group and associate options to it:

  1. In Content > Product > Options, click the +Add button at the bottom of the window to add an Option Group. Learn more tips and tricks for setting up option groups.

  2. Choose options as needed in the window that pops up:

    1. Option Group Name - Enter text here for the name of the Option Group.

    2. Create Unique Option Values (SKUs) - Select if you want the Site Manager to create a unique SKU for each option type.

    3. Option Type - Enter text in these fields for the types of options, (for example, color, size, length, and so on). Click the + Add button to add all the option types you'll need to create.

    4. Display Order - Choose the order in which you want the products to appear.

    5. Number of Values - For each Option Type, choose how many values will be associated with it. For example, if you are selling a t-shirt, you might have five sizes available.

  3. When you have set up all your option types, click the Continue button at the bottom of the page.

  4. The next page shows the options you have created for this product. If you need to make changes, you can return to the previous screen by clicking on the back arrow at the bottom of the window. When you are satisfied with the option types, click the Continue button at the bottom of the page.

  5. The next page shows every combination of option types you have created. If you need to make changes, you can return to the previous page by clicking on the back arrow at the bottom of the window. When you are satisfied with these option types, click the Save button at the bottom of the screen, and you'll see a message indicating that the Option Group has been set up successfully.

  6. In the main Options window, you will see the new Option Group you just created. In the Current Option Group area, you will see the Group Name and an Alias Name. Typically, the Group Name is what allows you to locate the Option Group in the Select Option Group drop-down menu; the Alias Name is frequently used on the front end of your site.

    Your site must be wired to accommodate Alias Name on the front end of your site. The Group Name and Alias Name can be the same, if relevant.

    If you want to change the Alias Name, simply type new text in the field and click the small Save button to the right of the field. To continue with more detailed settings, click the More Details link to continue.

 Editing an Option Group

When a product has options, many of the product details are set up at the option level. After you have created the various options for the Option Group, you will want to edit the remaining values for the product options. Once you have edited the rest of the values, click the save button at the bottom of the window after updating each option. The various option values are explained in the table below. Another way to edit an option group is explained after the table.

The UPC, Manufactured Country, Alt Part #, Bin #, Amazon SKU, ASIN, and Description are all optional if not relevant to your site or inventory management. If you are selling your products on Amazon, you must include the UPC, Manufactured Country, Amazon SKU, and ASIN.
Option Group options
Option Description

Display

Customer-facing display name associated with the option value, for example, red. Depending upon the wiring of your site (especially if selling on Amazon), the caption will need to be in one of the following formats if multiple options are associated with a single product. For two dropdowns on the product page use this format: Color=Red;Size=Small. For one dropdown, use: ColorandSize=RedSmall.

Value (SKU)

If you chose to let the system auto-generate the Unique Option Values (SKUs), this field will contain a unique SKU for each option type. This is the internal value that allows you to associate the product and option in order to discern which product to ship. Hyphens are the only special characters that can be used. All other special characters and spaces cannot be used. (no slashes, periods, spaces, symbols, and so on) This value can also be the same as what you enter in the Display field, if relevant.

UPC

The product's universal product code. This field is required if integrating with Amazon Seller Central.

Manufactured Country

This is the country where the product was manufactured. This is a required field if selling the product on Amazon.

Alternate Part#

A manufacturer’s part number, internal stock number, style number, or any other unique identifying number for this product at the option level.

Bin #

Example for use: if you warehouse your items and want to offer a more convenient way for your warehouse team to locate the product via the pick list.

Amazon SKU

Required for selling on Amazon. Amazon assigns unique product SKU numbers for items for sale on Amazon. If you know the Amazon SKU, enter it here.

ASIN

If you want to sell products on Amazon, enter the ASINAmazon Standard Identification Number: a unique identification number assigned by Amazon.com and its partners for product identification within the Amazon.com organization here. Each product sold on Amazon.com is given a unique ASIN.

Amazon FBA

Enter the Fulfillment by Amazon SKU.

Rakuten SKU

Unique SKU number used by Rakuten.com. Enter the number here if you want to sell the product on Rakuten.com.

Rakuten ID

Enter the Rakuten.com Listing ID, if known, here.

Description

If relevant, and your site is wired to accommodate, you may enter a description of the product option. If a customer selects the option on your site, the option description will replace the main image.

Inventory Status

Choose from the list of pre-determined inventory statuses such as: In-Stock, Closeout Special, Special Order Only, and so on.

Option Status

There are four product option statuses:

  • Archived Products will not be displayed on your website.
  • Discontinued Products will not be displayed on your website.
  • Incomplete Products is an available choice if you have not completed the setup for the product.
  • Live Products have been setup and are available for sale on your site.

Price Markup

The mark-up for this option over the product-level price. You can edit the price markup directly from this area. Click on Manage Price Breaks icon which opens the "Add Price Break" pop-up window. Here, you can set the option specific price breaks. You can also set these breaks for individual Tier Groups. In addition, the option level price breaks can be imported via spreadsheet. ​For example, you might want to use this feature if the option adds to the price of the product.

  • Quantity Break: If you want the price break to be given for over a certain quantity, enter it here.
  • ​Value: Enter the mark-up value in the Value field. For example, if you are selling shampoo, and the options are 8 oz., 12 oz., and 16 oz., the 12 oz. and 16 oz. options are likely to have a higher price than the 8 oz. product. In the Value field, enter the amount that this option adds to the price of the product. For example, if you have shirts in a number of sizes, you might want to mark up a large size because of your added costs.
  • ​Type: Select Fixed Adjustment to adjust the price for this option specified in the Value field. This can be a "+" or a "-" amount. Select Percentage to adjust the price by a percentage of the amount specified in the Value field.
  • ​Tier Group (Optional): If Tier Groups are in use on your site, you can set these price breaks for individual Tier Groups.
  • ​Apply to Option: Choose the product option(s) from the drop-down menu for which you want to add the price markup.
  • ​Display Price Adjustment as Separate Item with Label: Select this option to display the adjusted price separately and with a label.
  • ​Add: Click Add (+) to add the price markup to the option.

Cost

Use to identify the cost associated with the actual option, according to the same example as above. If you want to determine profit margins down to the option level, enter the cost for the option.

Quantity

For products with options, these option-level quantities will override any quantity settings at the main product level. Enter the quantity on hand of this option of the product. The quantity must be at least “1” to display in the shopping cart if your site is set to not allow more than on-hand to show. If a site is set to deduct inventory at checkout, make sure that your quantity field is regularly updated. If inventory deduction is enabled, the quantity is deducted based upon orders being placed by customers.

You will also need to set your Inventory Status and Option Status in order to have an option with a quantity of "0" not appear, if desired.

Display Order

Determine the order in which your options will display by entering a value in the Order field. If all values in the Order field are set to 0, the options will appear alphabetically. The master product option of the items is labeled “1.”

Wholesale Markup

If wired to do so, this field provides a Wholesale price for the product. Use this field if you will be offering a wholesale price to B2B customers versus a Sell Price for B2C customers.

Wholesale on Sale

If wired to do so, this field provides a Wholesale On Sale price for the product. Use this field if you will be offering a wholesale on sale price to B2B customers versus a On Sale Price for B2C customers.

Wholesale Sale Start

Useful if you hold sales on wholesale products that last for short periods of time. You can use this feature instead of changing your wholesale on sale prices regularly. The Wholesale Sale Start and Wholesale Sale End times set here will be displayed on your site only during the period of time set here. The time field uses the time zone you have set in your website settings. Start and end dates are optional, however if one is entered, the other is required.

Wholesale Sale End

Use the calendar menus and the time-picker to define when the Wholesale On Sale price will be applicable. If this field is populated, the Wholesale sale price remains on your site until you enter a valid end date and time or remove the wholesale sale pricing. Start and end dates are optional; however if one is entered, the other is required. The time field uses the time zone you have set in your website settings.

Regular Cost

Shows the regular cost of the product option.

Shipping Add-On

If you want to mark up the shipping rate for this option, the amount in this field will not override the shipping rate; it will only add this amount to the rate returned. If you want to markup the shipping on this option, enter the dollar amount of the markup.

Weight

Use Weight and Sub Weight to identify if the Option has a weight that adds to the overall weight of the product. Weight is the weight of the product in whole pounds or kilograms only. In the shampoo example described earlier, the 12 oz. and 16 oz. bottles would have a shipping weight greater than the 8 oz. bottle. As such, you will want to specify the weight differentials. If your site is set to US Measurements, this will be broken down into pounds. If your site is set to Metric Measurements, then you will use kilograms. You can set your default weight settings in the Users > Website Settings area in the weight measurement field.

Sub-Weight

Sub-weight of the product in ounces or grams. You can set your default weight settings in Users > Website Settings area in the weight measurement field. You may select either US measurements or metric measurements. In the shampoo example described earlier, the 12 oz. and 16 oz. bottles would have a shipping weight greater than the 8 oz. bottle. As such, you will want to specify the weight differentials.

ERP Low Threshold

Used for Amazon and Rakuten.com integration inventory management. If you set a ERP Low Threshold, when the product hits a quantity at or below the threshold, it will send a “0” as the quantity over to the channel. For example, if a red shirt has a quantity of “3” available, but the low threshold is set at “5,” a “0” will be passed across to Amazon to help against overselling merchandise.

ERP High Threshold

Used for Amazon and Rakuten.com integration inventory management. There may be times when you do not want to show the entire quantity available to the Amazon or Buy channels. Set the ERP High Threshold and those channels will not recognize any quantities higher than your setting here.

Oversized

You may also define if a specific Option changes the size of the oversized package by selecting the Oversized check box and adding values in the Width, Height, and Length fields.

Custom Freight

Select if this product option requires special shipping. If selected, this product option will ship using the custom freight you set up in Commerce > Shipping > Settings.

Fulfillment by Amazon

Select to indicate this product is available by Fulfillment by Amazon.

Non-Discountable Option

 

Image Option

If relevant, and your site is wired to accommodate, you can enter an image path (if the image is already in the ItemImages FTP folder) to the image for the option. This is a larger image than the swatch image. If a customer selects the option on your site, the option image will replace the main image. For example, if the option is green, when your customer chooses the green option, they will see the green option image.

Option Swatch

Swatches are typically smaller images intended to show variables such as colors or patterns. If your site is wired to accommodate swatches, select a swatch image from your FTP folder or desktop and upload it.

More about Option Groups

If many or all of the values are the same for each option in an Option Group, you may want to edit the options as described here. Once you have set up one option with all of the appropriate values, choose the + Add button to Create New Option. You can select the option you want to duplicate. When it is created, change the Display, Value (SKU), UPC, and Alternate Part Number (if applicable) for this option. Click the Save button at the bottom of the page to save your changes.

Tips & tricks

Aptos offers a few ways to make it easier for you to create new Option Groups that have similarities to other groups you have already created, as well as an ease of navigating between Option Groups.

Use the Select Option Group drop-down menu to find the Group Name you wish to view, edit or duplicate. Make sure you choose Option Group names that allow you to easily identify the group in this menu.

If you are creating multiple options with similar values, you can copy an entire group and then edit the values. To do this, follow these steps:

  1. Select the Option Group you wish to copy from the Select Option Group drop-down menu at the top right of the Current Option Group panel.
  2. In the Copy As field, enter the Group Name you wish to name the new group. Each Group Name should be unique.

  3. Click the Copy button.

    You will now see the new option group and options appearing on the page. Edit the options as needed. Make sure that you save both the group name with all the updated options by clicking Save Group icon at the top of the page and the Save button at the bottom of the page.

Use the Ctrl Key to select multiple choices.

 Seeing products assigned to an Option Group

You can quickly and easily see all the products assigned to an Option Group. To use this feature:

  1. Go to Content > Products > Options.

  2. Click the Binoculars icon and a pop-up window will open, showing all the products assigned to the current Option Group. The view is broken down by Sub-Category and shows each product in the Sub-Category.
