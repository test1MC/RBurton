Managing product personalizations

Personalization is a customized feature that can be used in different ways depending upon the wiring of your site. Typically, this is used if a product requires personalization (for example. engraving, monogramming, and so on). If product personalization is relevant to your site, this topic explains how to set up personalization questions and the format through which you will allow responses.

Setting up personalization

You will need to apply the Personalization option at the product level at Content > Products > Add/Edit > Details > Options to fully implement personalization.
Adding and Editing Personalization Groups
  1. In Content > Products > Personalization, enter a new group name in the Add Group ... field, and click the + Add button located near the top right of the window to add a new group.
  2. Click on the Add Personalization Option to begin adding and editing questions.

  3. Type the name of the group and save your settings. A new group will be created. In the example below, "Test group 4" has been created.

  4. If you create more than one group, in the Currently Managing field, use the drop-down menu to choose the group you will select on the Content > Products > Add/Edit screen.
 Adding Personalization Questions

You can create customized questions for each personalization group.

  1. In the Personalization Question fields, enter the questions that will appear on your site. This will prompt the customer to perform the action you set up in the Question Type area.
  2. From the Question Type menu, select the format in which you will accept responses to your question. For example, if you ask "What is your name?" You would set up the question type as a text box. (See definitions for the Question Types following the illustration below.)

  3. If the Question Type is editable, click the Edit Values link in the Personalization Type Values column. A pop-up window will open. You can set the Display Text and Lookup Value here. Add as many questions as needed. Click the Save button to save your settings.

  4. You can set a maximum number of characters for your text box entry points in the Max Length (Text fields only) field.
  5. Determine the order in which the questions will populate, if there is more than one, by setting the Display Order.

  6. To add new questions, click the + Add button beneath the Question Type field.

 Definitions for question types

Checkbox Options are a custom-defined list of check boxes for multiple selection values.

Question type definitions
Option Definition

Checkbox Options

A custom-defined list of check boxes for multiple selection values.

Click the Edit Values link. This will open a pop-up window in which you will define the values that appear in the applicable area on your site.

You can add and or edit the Display Text (what appears to the customer) and Lookup Value (what appears to the Site Administrator) by editing the content in the fields. Click the Add New Row link to add more questions.

Be sure to Save the values once completed.

Multi-Select List

A custom-defined list of check boxes for multiple selection values.

Radio Buttons

A custom defined list for single selection values.

Single Select List

A drop-down list of custom defined values in which a customer may only select one answer.

Multi-Row Textbox

A multi-row textbox for lengthy text spanning multiple lines.

 

Textbox

A single line textbox for short text values.
