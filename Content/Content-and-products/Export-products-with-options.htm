Exporting products with options

Use the Product with Options Export page to export product options at the product level.

Options and using the import/export spreadsheet are described in the document Using the Products with Options Spreadsheet. This guide defines all the columns in the spreadsheet, identifies required fields and explains the information needed for each field.

Using the import/export features in the platform is not necessary to build a website’s content. It simply provides an easier method to enter information and data. Aptos has two distinct types of products: Products with Options and Products without options. All products can be exported using the export products option, but it will contain only the information pertaining to the parent level product. All option level information must be updated using the Product Options drop-down option.

You should always first export this spreadsheet before you import products with options. Since the spreadsheet is updated from time to time, this ensures you are using the most recent version of the spreadsheet and will guide you on the proper format for each column and row, as well as which fields are required.

About requirements for the OptionValue field

Note these important requirements for the OptionValue field.

  • Do not use spaces in the OptionValue field.
  • Use unique, alpha-numeric values with no special characters: no slashes, periods, spaces, symbols. A hyphen is the only exception to this rule.

Examples of acceptable text:

  • color-blue
  • color=blue
  • size=large

To export product with options to spreadsheet

  1. In Content, select Import/Export > Product With Options Export.
  2. Choose whether to Export Products By either individual Sub-Categories, Brands or Option Groups using the list.
  3. Select the brands or categories for which you want to export product information. Select more than one by holding down the Ctrl key. You can also use the Select All and Clear All links.
  4. Use the check boxes labeled Include to select what to include in the file:

    1. products in shared sub-categories

    2. deleted products

    3. inactive products

    4. inventory only

  5. Click the Export to Excel button .

    A pop-up window opens showing details of the scheduled export.

  6. Make any changes needed to the export schedule date and time, and notification email address if needed, and click Save.

    After the export has run and completed, a link to it appears at the bottom of the main screen.

    You may need to refresh your browser window for the link to appear.

  7. creates an Excel spreadsheet and a link to it at the bottom of the page. Click this link to open, modify, and or save it. Product information spreadsheet fields you can modify are:
