Fulfilling orders for pickup

Work through these procedures to fulfill pickup orders. Begin by accessing the Order Worklist; use the shortcut or link you were provided with. Orders are listed in order of oldest to newest, but you can toggle between listing oldest and newest first by clicking the Elapsed Time column header.

Pick up orders are indicated by the text Customer Pick Up in the Destination column.

Note that the worklist is dynamic, and is updated as new orders come in. You’ll see these new orders when you return to the Order Worklist, or refresh its screen. Expedited orders are shown in red.

1. Generate and print order pick lists

  1. Use the check boxes to select orders for pickup—denoted by the text Customer Pick Up in the Destination column—you want to work on. If you want to work on all of them, filter by Pickup New, and click the check box at the top of the column.

  2. Click the Print Selected button above the Order list.

    This generates a collated PDF of individual pick lists for each order selected, and displays them as a single PDF document. Item statuses are set to Acknowledged, and the fulfillment status is set to In Progress.

  3. Print the PDF pick lists. Exactly how to do this will depend on the specific browser and operating system you’re using.

  4. Click your browser’s back button to return to the Order Worklist. Continue to the Fulfill pickup orders procedure.

2. Fulfill pickup orders

  1. Use the pick lists you printed to collect merchandise for the orders you’re fulfilling.

  2. Return to the Order Worklist.

    If your session has timed out, get back to the worklist using the shortcut or bookmark you normally use to go to it, rather than using the login screen to provide a user name and password.

  3. In the Order Worklist, find an order for which you’ve printed a pick list (using the Search field to locate it if necessary), and click its order number. You’re taken to the Order Worklist fulfill orders page for that order, showing all items in it.

  4. Confirm items you are ready to fulfill. Click the Click Here to Scan button if needed; otherwise, the text Ready to Scan appears. Then either::

    1. If you have a Keyboard Wedge scanner, scan a product’s UPC. With each successful scan, the item count for that product increments. Or …

    2. If you don’t have a scanner, or if the UPC label is damaged, manually enter the product UPC in the field provided, and press Enter. Watch as the item count increments. Or …

    3. If enabled for your stores, you can click the + button to add items. You can also use the– button to subtract items, until correct.

      Note: Click the Can’t Fulfill Remaining button for any item you can’t completely fulfill, and select a reason from the list that’s presented. That line is then canceled from the order.

  5. When all items in the order have been successfully selected or canceled, click the Ready for Pickup link.

    If you’re unable to fulfill any items in the pick up order, click Save Changes, and you’re returned to the Order Worklist.

    EOM sends an email to the customer, letting them know their order is ready to pick up at your store. You’re taken back to the Order Worklist, with any filters you’ve engaged still active, where you can move on to the next order to fulfill.
