Managing harmonization codes

In the U.S., HSHarmonization System numbers are also called Schedule B and HS Tariff Classification numbers. To find tariff information for exports of U.S. products, you must identify the HS number for your product up to the six-digit level. If possible, it is recommended that you classify your product to the eight-digit level to ensure proper duty identification.

Due to the number of possible harmonization codes (>20K), no harmonization codes are displayed upon initial page load.

To add a harmonization code

  1. In Commerce, select Shipping > Harmonization Codes.

  2. Click the Add button (plus sign).
  3. In the Add Harmonization dialog box, enter the new Code and Description for it.
  4. You can override the standard description by entering custom description in the Override Description field. When the modified harmonization code is used elsewhere in the application, it uses this description rather than the standard description.
  5. Click Save.
 To search for a harmonization code

  1. In Commerce, select Shipping > Harmonization Codes.

  2. Search for the code you want to see; search by typing text in any combination of the fields.

    You can enter partial text in these fields; the search engine uses what you enter to find codes containing that text.

  3. Click the Search button.
