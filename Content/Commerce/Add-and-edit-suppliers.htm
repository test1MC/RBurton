Adding and editing suppliers

Use this page to manage suppliers, and to import and export suppliers to from/to a spreadsheet.

Note about Stores: Individual store locations are treated as Suppliers for order fulfillment. Stores are treated as Affiliates for order origination, and as Locations for store locator pages.

To search for a supplier

  1. In Commerce, select Suppliers > Add/Edit.

  2. Type the Supplier Name in the search field, and select the Supplier Group if applicable from the list.

  3. Choose whether to Include inactive suppliers in search results.

  4. Click Search.
 To add or edit suppliers
  1. In Commerce, select Suppliers > Add/Edit.

  2. Click Add Supplier to add a new supplier; this opens the edit page where you can add details about this supplier. See information in Add/Edit fields, Supplier settings and Drop ship settings tables for details.

  3. To edit details for an existing supplier, click its Edit button.
  4. Click Save.
Add/Edit fields
Field Description
Supplier ID A unique ID automatically generated by the system. It is used by the system to identify the supplier's record in the database.
Supplier Name Required; the supplier's name. This name will be used as the display name in the site manager.
Display Name Required; the Supplier's company name. This is the name displayed on the store's front-end.
Address Required; the supplier's street address.
City, State, Zip Required; the supplier's city, state, and zip code. Note that the state is the two character abbreviated format.
Phone The supplier's phone number.
Email Address Required; the supplier's email address. Used to identify a supplier in .
Account Number The Account Number used to identify a supplier account in one or more software applications.
User Name Required; a unique name used for display purposes in the site manager.
Password Required; a unique password that a supplier will use to login to their site manager account.
Supplier settings
Field or button Description
Active On/Off slide switch makes the supplier active or inactive in . Disabling a supplier account prevents the supplier from logging into the site manager; this does not delete the supplier account.
Priority Set if you want to prioritize or rank your suppliers. If desired, number your top suppliers so they will be given first priority when fulfills orders.
Expedited Shipping Cutoff Time The cutoff time for expedited shipping.
Supplier Group The supplier group for the supplier if applicable. Currently, supplier groups can only be managed via a spreadsheet import.
PO Box Shipping Select On to allow the supplier to ship to PO boxes.
PO Signature Required Select On if you require a supplier to sign purchase orders. This is a means for authenticating the contractual nature of the purchase order.
In-Store Pickup

If On, the supplier (store) will be acting as a point for in-store (local) pickup. Two fields are displayed:

  • Edit the text field for the Checkout availability message (In-store pickup) if needed. You can customize this message so that when a customer places an order from a store marked for in-store pickup or if it is providing drop-ship services, the customer will immediately know the availability of the items in their cart.
  • In the second field, enter specific terms of pickup for this location. This text will appear on the order confirmation page and order confirmation email.
Select Available Brands Click this link to assign specific brands to the supplier. Only the brands that have previously been added to the system are available to select.
Drop ship settings
Field or button Description
Drop Shipping Select On if the supplier provides services for drop-shipping.
Offers Rebates Select On if you want to allow special offers and rebates.
Payment Terms Payment terms between you and the supplier. If you do not see the desired payment terms listed, contact your Client Manager to discuss scoping.
Select Taxable States If you use 's built-in sales tax functionality, click this link and select the states that collect sales tax.
Syncing suppliers from locations

In cases where store locations may be fulfilling orders, you can enter those locations here once addresses have been entered.

To sync suppliers from locations

  1. Click the Sync Suppliers from Locations List link at the top of the Add/Edit Suppliers window.

    A new window opens from which you to select each location to be synced as a supplier.

  2. Select the Sync? check box for each supplier you want to sync to their locations. The location's "State" field value will be pulled in as the two digit abbreviation for that state.
  3. ​If the supplier locations are designated where pick lists, packing slips, and shipping labels will be printed, select Setup Suppliers as a Printing Station at the bottom of the page.
  4. Click Save when setup is complete.
  5. Click the Return button to return to Add/Edit Suppliers window.
 To email suppliers

You can email all suppliers directly from the Add/Edit Suppliers window. The email window consists of a WYSIWYG editor that supports HTML, tokens, and the ability to attach files.

  1. Select the Email Suppliers link at the top of the Add/Edit Suppliers window.
  2. In the Email Your Suppliers email editor, enter a subject line and body for the email. Attach files as needed.
  3. When ready to send the email, click the Send button at the bottom of the window.
 To import/export suppliers

In addition to adding suppliers manually, you can also import or export suppliers using a spreadsheet. See Content > Import/Export > Suppliers.
