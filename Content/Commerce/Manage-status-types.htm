Managing Status Types

Use this section to view and manage the order status types in use on your site.

The order status types activated for use on your site are displayed on this page.

  • Status ID: This ID number is associated with the order status name. If you change the name of the order status, the ID will remain the same.
  • Status Name: The default order status names are displayed here. While you can change these names for business purposes, it is not recommended. If you change names, the status ID number will remain the same and will be associated to the default order status name.

Locks Edit Order Tab

You can control whether you want to manage order statuses at the order level or the item level. Configure control for which column you want to use at Users > Website Settings > Site Layout > Website Settings.

  • Lock Edit Order Tab at Order Level Status - If this column is enabled, use the check boxes to control the statuses you want to lock at the order level.
  • Lock Edit Order Tab at Item Level Status - If this column is enabled, use the check boxes to control the statuses you want to lock at the item level.

The Ordered Status Type check boxes will always be enabled for both Item and Order Level Statuses.
