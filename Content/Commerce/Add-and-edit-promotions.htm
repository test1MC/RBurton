Adding and editing stackable promotions

EOM gives you the ability to add, edit, review or discontinue current site promotions. Here you'll learn how to create and manage stackable promotion codes and campaigns.

You add edit and manage stackable promotions in Commerce > Promotions > Add/Edit

To search for current promotions

To search for and find promotions, you can filter by the promo name, promo type, deleted promos hidden promos, and by auto triggers. Select search options to refine your search if needed, and click the Search button . Current Promotion Codes, if any, will be displayed.

Search promotions options
Option Description
Promotion Name Promotion name is another way of saying promotion code. This will either be custom created or automatically generated. (for example, 10OFFALL or 272dW7e)
Promotion Type To filter by a promotion type, select it from the drop-down menu.
Show Deleted Searches for a deleted promotion; you can then restore it, if needed.
Auto Triggers Searches for promotions that auto-trigger in the cart rather than requiring the customer to enter the promo code.

Current promotion codes

The Current Promotion Codes area displays the codes that have already been configured. The columns display:

  • Promotion Type – The type of promotion used along with the code name of the promotion.
  • Dates – The date the promotion was created, the beginning date and the expiration date.
Some of the columns displayed do not function with stackable promotion codes. Only the columns described here work with stackable promotion codes.

Creating promotions and promotion codes

  1. From Commerce > Promotions > Add/Edit, click the “+” (Add) button. The Add New Promotion Code window opens.
  2. Create unique promotion codes and set the parameters of a new promotion here. There are six different stackable promotion types to choose from. These are:

    1. Item Dollar – Gives dollar discount off an item. (Item level.)

    2. Item Percentage – Gives percentage discount off an item. (Item level.)

    3. Order Percentage – Gives percent discount off an order total. (Order level.)

    4. Gift with Purchase – Gift for orders over a set dollar amount. (Order level.)

    5. Free shipping – Gives free shipping on orders over a dollar amount. (Shipping level.)

    6. Subcategory BOGO Percentage – Buy one of an item and get percentage off second item. (Item level.)

      Each of these and their settings is described below.

  3. When configured, click Save. This new promotion code will be displayed on the Current Promotion Codes area of the page.

Promotion types

Available promotion types and their settings are:

Item dollar item-level promotion

Use this item-level promotion type to apply a specific dollar amount discount to selected items, using the product ID/SKU.

Setting Description

Promotion Type

Select the Item Dollar promotion type from the list.

Promotion Code

Give the promotion code a unique name or click the Generate Code link and one will be automatically generated.

Promotion Short Description

It helps to have a description of the promotion, especially for CSRCustomer Service Representative information. (For example, Free Silk Dress with Purchase over $250).

Dollar Discount Amount

Enter the dollar discount amount in this field. For example, enter 10.00 to give a $10 discount off the price of the items.

Valid Between

Start: Enter the start date and time manually or by clicking on the calendar and time picker icons.

End: Enter the end date and time manually or by clicking on the calendar and time picker icons.

Items Receiving Promotion

Select the Get Item link to make your selections for the items you want to receive the promotion. This will open a dialog box. You can search for and select products by sub-category, brand or product name. Select the products you want, and click (+ Add Select Products) at the bottom of the dialog box to save the settings. The choices you make here are automatically populated in Items Receiving Promotion field.

How does this promotion trigger?

Use the radio buttons to select one of these options.

  • Auto-triggers in cart. This promotion code will automatically trigger and display in the shopping cart.
  • User must enter promotion code. Requires the customer to know and enter the promotion code manually in the shopping cart.
Only one item level promotion can be applied at the same time. If more than one promotion applies, the promotion with the earliest start date will be used. If the promotion start dates are the same, the one with the newest create date will be used. If the create dates are the same, the lowest promotion ID will be used. If a manual promotion code is applied to the cart but one or many items it applies to have an automatic promotion, the manual promotion will be honored over the automatic promotion.

This promotion is triggered by configuring sub-categories. So, if the customer’s cart contains valid products from the chosen sub-categories, the promo will be triggered and they will receive the discount.
