Managing saved orders

You can save orders for your customers, if relevant to your business. If Save Proposal was selected when creating an order at Commerce > Orders > Create New, is saved and viewable here at Commerce > Orders > Saved Orders.

To manage saved orders

  1. In Commerce, select Orders > Saved Orders.

  2. Use the search functionality to locate the order.

    Note: You can use any of the fields listed, or add a search condition to find a saved order.

  3. Locate and select the order using the radio button in the left-most column. Scroll to the bottom of the screen if necessary; now you can either:

    1. click the Select the ">" Retail Order button.

      This takes you to the Commerce > Orders > Create New screen. You can make edits if necessary and proceed with processing the order by clicking the Process Order button at the bottom of the screen.

    2. Or, if the order is no longer applicable, you can delete it by clicking delete button.
  4. To export orders to an Excel spreadsheet for reporting or printing, click the Export To Excel button .
