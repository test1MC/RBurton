Adding and editing affiliates

To create a new affiliate

  1. In Commerce > Affiliates > Add/Edit, select No Group from the Select Affiliate Group menu.

  2. Click the + Add button at the bottom of the page.

  3. Complete the information in the Add Affiliate registration pop-up window. Fields and options are explained in the table below this illustration.

  4. Save.

To edit an affiliate

  1. In Commerce > Affiliates > Add/Edit, select Affilitate from the Select Affiliate Group menu. A list of all affiliates appears. You can edit and manage the Affiliate from here. You can also search for a specific Affiliate by pressing CTRL-F on your keyboard, type the name of the affiliate you're searching for and your browser will return search results.

  2. Make edits as needed.

    1. ID - The new Affiliate code will be located on the left side of the screen under ID. This code is used for any landing page, promotional email, or other marketing material for which you would like to track information back to that Affiliate.

      1. To add your code to a URL: Add “/?aff=#### ” to the end of your URL.

        Example: www.Aptos.com/?aff=5130

        1. For an Affiliate to get credit for an order, the Affiliate must to be associated to the order. To do this, the Affiliate ID will be added to the query string of the site. For example, there is an Affiliate in production called "info" that has an Affiliate ID of 5107 so the URL for the Affiliate for the site would be "www.yoursite?aff=5107. This will store a cookie on the browser of the Affiliate and will be used to associate the order to this specific Affiliate.
    2. Edit - To edit the details of an Affiliate, select the Edit icon in the Edit column. You can edit any fields in the pop-up window. Click Save after you have made your edits. The Affiliate will be updated with the edits you just made.

    3. Status - You can set the Affiliate status to Active or Inactive.

    4. Cookie Life - Enter the number of days that the cookie will be active.

    5. Group - Shows the group the Affiliate belongs to.

    6. Costs - Enter information here if there are costs associated with the Affiliate.

      1. Cost/Acquisition - Enter an amount here if there is a cost per acquisition. In other words, enter an amount that you will owe this Affiliate when an order is placed. This can be a dollar amount or a percentage.

      2. Cost is Flat Fee - Select this box if the acquisition cost is a flat fee.

      3. Cost/Click - Enter an amount if there is a cost per click associated with this affiliate.

    7. URL - The URL of the affiliate's site.

    8. Statistics - Click the Statistics icon and you can enter a Date From and Date Thru range and select an Order Status from the drop-down menu to view a breakdown of statistics for that Affiliate over that specific time period. The breakdown includes: Visit Date, Visits, CPC, Total CPC, Orders, Total Revenue, CPA, and Total CPA. Once the report has populated, you can export it to an Excel file by clicking on the Export icon at the bottom of the pop-up window.

  3. Save.
 To import and export affiliates

You can quickly import or export affiliates, using a spreadsheet on the Affiliates page.

  1. In Commerce, select Affiliates > Add/Edit.

  2. Click the Import/Export Affiliates menu bar.

    The page expands to show the Import/Export Affiliates area.

  3. To export and edit affiliates:

    1. click the Export to Excel icon. This creates an Excel spreadsheet and a link to it beneath the Import/Export buttons.

      You should always export first to ensure you are using the most recent spreadsheet version. This will also guide you on the proper format for each column and row as well as which fields are required. An exported spreadsheet will be similar to this one:

    2. Click the spreadsheet link, add or update information in it, and save the spreadsheet to a local drive.
  4. To import affiliates:
    1. In the Import/Export Affiliates area, click the Choose File button.
    2. Browse to and choose the Excel file you edited and click Open.
    3. When the name of the file you chose appears in the Excel file address field, click the Import button to upload new affiliate information in the spreadsheet.
  5. Click Save.
