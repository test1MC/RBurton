Creating a new order

The Create New order screen streamlines the order-taking process and improves the way everyone—and every system—in your company interacts with the platform's order management system. Orders can be easily created in the site manager by Customer Service Representatives (CSRs) in a call center environment, and sales associates using a POS in store. The screen resembles the New Order screen on the front of your website.

The Create New screen enables quick order processing. Access to various ordering functions is available in tabs across the top of the page.

These tabs contain all the actions needed to process an order.

  • Customer Search contains three search filters, with the ability to add more if needed. Search results are displayed inline at the bottom of the page. You can quickly create new customer accounts on the main Customer Search page.
  • Add Items to Order uses the same filters displayed on the front of your site. Finding products to complete an order can be done quickly and easily.
  • Shipping lets you ship items to multiple addresses in the same order; you can also create separate gift messaging for each item. Pick up in store functionality is also configurable from the Shipping tab.
  • Payment Info is easy to manage in its own tab.
  • Order Summary is located in a quick view tab which eliminates unnecessary scrolling.
  • Current Order information is located in a persistent display on the right side of every page.

To create a new order

1. Find the customer

Before you create an order, search for the customer in this area. If the customer is not in the system, you can quickly add them here.

  1. In Commerce, select Orders > Create New.

  2. Enter search information in any of the search fields: Name, Zipcode, or Email Address. You can also add an additional search field by clicking Add Search Field, and selecting one you'd like to use.

  3. Click Search.

    Search results are displayed below the search fields.

  4. If the search results don't include the customer you're looking for, you can create a new account for them; see To Create a new account.

  5. In the search results, select the customer you're looking for using the radio button for that customer. The customer's information auto-populates in the billing and shipping fields.

  6. After you've selected a customer, customer notes are displayed at the top of the page. You can quickly add to the customer notes displayed here if you need to.

To create a new account if needed

If a customer is not in the system, you can quickly add them here. You can add billing and shipping information later.

  1. In the Search Results record fields, add the information:
    1. Company if applicable
    2.  Customer Name
    3. Customer Email
    4. Phone
    5. Tier Group if applicable.
  2. Click Create New Account. The new customer is added to the system.
 2. Add items to an order

The Add Items to Order tab uses the same filters that appear on the front of your site. You can quickly find a product to add to an order using the built-in filters.

To add items to the order
  1. With the new or existing customer selected, click the Add Items to Order tab.

  2. Use the filters to search for products or type in the item name or SKU in the search field. The search results display under the search filters complete with the image, product options, and the price.

    If set up on your site, this is also where options such as subscriptions options and personalization options are displayed for items that have those configurations.

  3. Select the options and quantity, then click Add to Order.

    The Current Order information displays in a persistent area on the right side of every page.

  4. At any time during order processing, you can add a Promo Code to the order, and click Apply. The promo code discount is deducted from the order total and displayed at the bottom right of the page.
 To modify an order

You can also modify items after they have been placed in the cart.

  • Change the quality and click Update to change the quantity of an individual item in the order.
  • Select the Edit Options link if you want to change selected options for an item. After you have made the changes, click Save Options.
  • Delete an item in the order by clicking on the red X.
 To add cross-sells

If there are cross-sells available for any products, the Cross-Sells Available panel is displayed at the bottom of the page.

  1. Click to expand the panel and view the cross-sell items.
  2. Click Add to Order to add these items to the order.
 To add items from saved carts

If a customer has saved items to a shopping cart, the Saved Cart tab is displayed.

To add any of these items to a new order, select the applicable options and click Add to Order. The item is added to the current order.

 To add items from a wishlist

If a customer has created a wishlist in the member's area on your site and wants to add those items to their order, access it from the Wishlist tab.

To add any of these to a new order, select the options if applicable, and click Add to Order. The wishlist item is added to the order.

 To add items from previous orders

If this is a returning customer, you can access their previous orders from the Previous Orders tab. Each previous order appears in a separate collapsing panel.

  1. If a customer wants to reorder an item, open a previous order, and the items in that order are displayed.
  2. Change any options as needed and click on Add to Order to add an item to the new order.
 3. Add shipping information

All shipping information is located in the Shipping tab. Note that the current order information is displayed on the page as well.

The 'ship to' functionality is incorporated on the page including:

To ship to one address

When shipping to only one address, add the customer's shipping information in this area.

  1. Enter or verify the customer's shipping address; all fields marked with an asterisk * are required:
    1. If this is a first-time customer, add the billing and shipping information.
    2. If this is a returning customer, the Ship To list is populated with the customer's billing and shipping addresses already in .

  2. Select whether the address is Residential or Business.

    This information is used by , along with the address, to return the correct shipping rates.
  3. In the Standard Shipping area, select the the shipping option the customer prefers.

  4. In the Override Shipping Amount field, enter a dollar value if you want to manually override the shipping amount.

    You can select any shipping service supported by , and manually enter the shipping rate. If supported by your site, you can select the ManualThe Manual feature is only available if Hide Edit Shipping website setting is disabled and the Select manual radio button by default website setting is enabled. radio button and define the appropriate shipping method and rate without needing to call the carrier to get ratesThis is useful if shipping a large product volume using an express shipping carrier..

  5. If the customer wants to wait until all the items in the order are available before they are shipped, select All orders must be held to ship complete. If selected, all items will ship together in one order.

  6. To add a gift message to the order, select Add a gift message, and enter the message in the field that appears.
 To ship to multiple addresses

When shipping items in an order to more than one address, select the Ship to multiple addresses tab. Each item in an order is displayed on its own line so that every item can be shipped to a different address.

  1. For each item in the order, click the Ship To list and select an existing address, if applicable.
  2. If an address is not in , select + Add New Address and complete and save the shipping address information.
  3. To add a gift message to individual items in the order, select Add a gift message, and enter the message in the field that appears.
 Pick up in store (if applicable on your site)

If you have physical store locations, and this is configured on your site, select the Pick up in store tab if the customer wants to pick up their purchase at one of your stores.

  1. The stores closest to the customer's location are shown when you select the Pick up in store tab.
    1. To select stores near a chosen zip code, enter the customer's zip code and select Change Location.
    2. The item availability and store location and details are displayed on this page.
  2. Tell the customer which stores have the product in stock. Select the store the customer chooses.
  3. Click Continue. The store is notified and automatically sends an email to the customer when the item is ready to be picked up.
 4. Add payment info

Define payment information in the Payment Info tab. The current order information is displayed on this page as well.

  1. If this is a first-time customer, add their payment information here. If they are a returning customer, their payment information will be in the system if it has been saved.

    Depending on how your site is configured, and the payment types your company accepts, these payment types may be displayed.
    1. Credit/Debit—American Express, Discover, MasterCard, or Visa
    2. PayPal
    3. Purchase Order
    4. Cash—Wells Fargo, Help Card, Electronic Check, Money Order, or Split Payment
  2. Select the tab for the payment type the customer wants to use for this order. You can choose to save the customer's payment information.
  3. Select Purchase Order Number if the customer would like to reference a purchase order number.
  4. Select My billing address is different than my shipping address if needed, and add a new address or select an existing address.
Aurus integration

If your company uses the Aurus credit card processor to authenticate credit cards and capture funds, this page will look a little different than described above. This is because this screen incorporates an iframeinline frame, basically an embedded web page supplied by an external source sourced directly and dynamically from Aurus . Exactly how this screen looks may also change from time to time due to updates from Aurus.

To authorize a credit card with Aurus

  1. Enter the Card number, card verification number, and select the card's expiration month and year.
  2. Enter the card holder's name.
  3. Click the Start Authorization Process button. Button text may differ.

This sends an authorization request to Aurus. An error message is displayed if the card is declined.

Supported Aurus transactions

As of the 2018.2 release, Aurus does not yet support:

  • Apple Pay
  • PayPal
  • gift cards
  • Use of the Credit Card Terminal.

These will be supported in Aurus in future releases.

 5. Order summary

The Order Summary page displays a finalized view of billing, shipping, and payment information. You can also add order notes and various marketing functions here as well.

  1. Add Order Notes to the order as needed.
  2. Choose Additional Options:
    1. Select Do not apply sales tax as needed for business purposes.
    2. Select Do not charge eWaste according to your business practices.
  3. Select the AffiliateDenotes whether the order came through natural search, Google Products, Commissions Junction, Rakuten, and so on. to assign to the order from the list if this is set up for you.

  4. Select the type of payment processing depending on your business practices:

    1. Capture Funds at Checkout

    2. Manually Process Checkout

    3. Authorize Only at Checkout

    4. Authorize and Void at Checkout

    5. Auth/Void Fixed Amount, Capture Actual

  5. At any time during the order processing, you can add a Promo Code from the list, and click Apply to apply it to the order.

    This page also accepts multiple promo codes, but only if the setting is turned on. The confirmation email for orders placed in the site manager will show the multiple promo codes used on an order.

  6. Select any Opt-In Preferences your customer wants. All are all selected by default.

  7. Place order or save for later:

    1. Click Save for Later if the customer wants to wait to place the order, and revisit it later.

      To allow customers to see proposals, enable the feature in Users > Website Settings > Site Layout > Website Settings. Go to Allow Customer to Open Proposals as Saved Cart and select this feature.

    2. Select Place Order to complete the order. You'll see a confirmation that the order was placed successfully.

About Wells Fargo payment options

If your company accepts Wells Fargo as a payment option, there are additional steps required. The Wells Fargo financing option is only available if the minimum dollar threshold is met. If available for the order, the Wells Fargo Financing button is displayed. Click the Wells Fargo Financing button and a new pop-up window appears, prompting you to either Apply Now for credit, or use an existing line of credit.

Wells Fargo payment options such as the minimum dollar threshold, are set up in Users > Website Settings > Payments. This website setting is called "Use Wells Fargo in Retail Order Screen."

From this point on, the process is the same as on the front end of the site. When viewing an order, the Wells Fargo information appears in the Order Notes in the Order History.
