Managing tax categories

Tax Categories are the highest level of tax data, which indicate the broad categories that each product will fall under. You can select from three options for your site's sales tax settings. You can use Aptos's tax rules, Avalara for address verification (AVS), or fully integrate with Avalara's AvaTax™ service which enables you to manage rules and sales tax codes for address verification and/or tax calculations. Learn more at Users > Website Settings > Payments.

This page in the Site Manager allows you to manage two tax functions: Tax Categories and Tax Category State Details rates respectively.

To use this functionality, you must first ensure that the tax settings are enabled. To do so, go to Users > Website Settings > Payments and select Calculate Sales Tax at Zip/Postal Code Level and Override Zip/Postal Code Tax Lookup with Category Tax.

To add a tax category

  1. In Commerce > Settings > Tax Categories, click the Add Tax Category link to add a new tax category, and provide details for it:
    1. Category Name - Enter the name of the tax category.
    2. Category Code - Enter the shortened code name of the tax category.
  2. Click the Save button at the bottom of the page. A confirmation message appears after the update has it has been successfully saved.

  3. You can continue to add as many tax categories as needed for your site. You can also modify and delete tax categories as needed.

About tax category state details

Use this functionality to edit sales tax percentages for the states in which you must collect sales tax. When using Aptos tax rules, the system will calculate sales tax based upon ZIP codes. Sales tax tables from Tax Data Systems are updated on a monthly basis. To use this feature, go to Users > Website Settings > Payments and select Calculate Sales Tax at Zip/Postal Code Level and Override Zip/Postal Code Tax Lookup with Category Tax.

Tax Category State Details rates section is where you set up US States per tax category, with the accompanying tax rate and whether shipping and/or handling need to be taxed for each state.

To add new state detail tax rates (one at a time)
  1. In Commerce > Settings > Tax Categories, Tax Category State Details, use the drop-down menus to select the state and tax category (set up in the section above). Select the tax rate by using the up and down arrows highlighted in the illustration below. Choose whether or not shipping and handling will be taxed.
  2. Click the + Add button. The state and the settings will appear in the table below.

  3. Save the setting. Repeat for each state you set up, saving each state's settings before setting up the next state.

 Importing / exporting Excel files

To import an Excel file in the Tax Category State details section, the file must be correctly formatted. Aptos recommends that you first export the spreadsheet to ensure you are using the most recent version before editing and importing the file.

The first row column headers are:

  • Category Name - The tax category name. When creating a file, it is important to include this header row in the file.
  • State Code - US State Code (2 characters)
  • Tax Rate - Decimal rate representing the tax rate for that State/ Tax Category
  • Shipping Taxed Y/N - 1 character, Y or N
  • Handling Taxed Y/N - 1 character , Y or N
  • Delete (Blank/True/False) - Indicates whether or not to delete the record. Leave blank for no change.

About product tax category administration

After tax categories have been set up, you can proceed to assign individual products to their particular Tax Category for specific states if needed. Certain products may be taxed differently in various states. If the product is not assigned to a Tax Category, the product will not be taxed in that particular state. You will want to edit the information here for a specific product. If, for example, a state charges different rates for food vs. clothing; and you sell t-shirts and candy bars, you can edit the tax information for different products at Content > Product > Add/Edit > [product] Details > Attributes > Tax Category Management.

To set up tax codes at the product level, work with your Client Manager to dedicate a field for storing tax codes. When you have finished, click to save your changes.
