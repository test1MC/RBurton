Fulfilling orders for shipping

Work through these procedures to fulfill shipping orders. Begin by accessing the Order Worklist; use the shortcut or link you were provided with. Orders are listed in order of oldest to newest.Orders are listed in order of oldest to newest, but you can toggle between listing oldest and newest first by clicking the Elapsed Time column heading. This number represents how long an order has been active.

Note: Shipping orders display a shipping address in the Destination column; pick up orders are indicated by the text Customer Pick Up in this column.

Note that the worklist is dynamic, and is updated as new orders come in. You’ll see these new orders when you return to the Order Worklist, or refresh its screen.

1. Generate and print order pick lists

  1. Use the checkboxes to select orders for shipping you want to work on. If you want to work on all of them, filter by Ship New only, and select the check box at the top of the column.

  2. Click the Print Selected button above the Order list.

    This generates a collated PDF of individual pick lists for each order selected, and displays them as a single PDF document. Item statuses are set to Acknowledged, and the fulfillment status is set to In Progress.

  3. Print the PDF pick lists. Exactly how to do this will depend on the specific browser and operating system you’re using.
  4. Click your browser’s back button to return to the Order Worklist. Continue to the Fulfill shipping orders procedure.
 2. Fulfill shipping orders
  1. Use the pick lists you printed to collect merchandise for the orders you’re fulfilling.
  2. Return to the Order Worklist.

    If your session has timed out, get back to the worklist using the shortcut or bookmark you normally use to go to it, rather than using the login screen to provide a user name and password.

  3. In the Order Worklist, find an order for which you’ve printed a pick list (using the Search field to locate it if necessary), and click its order number. You’re taken to the Order Worklist fulfill orders page for that order, showing all items in it.

  4. Confirm the items you are ready to ship. Click the Click Here to Scan button if needed; otherwise, the text Ready to Scan appears. Then either:

    1. If you have a Keyboard Wedge scanner, scan a product’s UPC. With each successful scan, the item count for that product increments. Or …

    2. If you don’t have a scanner, or if the UPC label is damaged, manually enter the product UPC in the field provided, and pressEnter . Watch as the item count increments. Or …

    3. If enabled for your stores, you can click the + button to add items. You can also use the – button to subtract items, until correct.
    Note: Click the Can’t Fulfill Remaining button for any item you can’t completely fulfill, and select a reason from the list that’s presented. The Order Routing system then tries to find another store that can fulfill that line item in the order.
  5. When all items in the order have been successfully selected or re-routed, click the Proceed to Ship link. You’re brought to the Ship Order screen. Continue to the Ship orders procedure.
 3. Ship orders

  1. In the Ship Order screen, use item check boxes to select items you want to ship.

    Note: If you need to pack items in more than one box due to size or weight restrictions, you’ll have to perform this procedure more than once (see step 5 below).

  2. Select appropriate Package Size for the items selected; Shipment Weight is auto populated based on the selected items.

  3. Confirm the Carrier and the Service. You may or may not have the ability to change this based on your company’s configuration.

    Note: You don’t need to fill in the tracking number field, as it auto-populates after the shipping label is generated.

  4. Click the Pack and Ship Items button. This generates shipping labels and packing slips for the box. Click individual links to display and print slips and labels as needed.

    Note: Depending on how fulfillment is configured for your company, not all documents shown below may be available. For example, you may not see a Return Label link, and its associated Return info section in the Packed Items area of the screen. Or, again depending on your company’s configuration, a default document to print may appear in the printer dialogue immediately.

    Tracking numbers now appear for those items.

  5. Any items not selected for shipping in step 2 above appear in a list above the packed items; repeat the process for any remaining items

  6. Click the < Worklist link to go back to the Order Worklist, and continue to fulfill orders; note that any filters you used are still active.
