Fulfillment for Store Associates

This section describes the order fulfillment process for store associates. It provides basic workflows used for orders shipping to customers, as well as for customer pick up orders. Special circumstances, for example partial pick ups, are also covered. First, here are a few basics.

Gaining access to the Order Worklist

As a store associate, you’ll be given a browser shortcut or link which automatically logs you into Site Manager, and, brings you to your Order Worklist showing orders assigned to you and your store, ready for fulfilling.

Be sure to bookmark this shortcut or link, and always use it to regain access to the Worklist, especially when the system automatically logs you out due to inactivity. If it does log you out, it may ask you to log back in. Rather than attempting to do that (using a login name and password), use the shortcut you were given.

 About filters

You won’t need to do any special filtering to show orders needing your attention. If you do, however, want to just see certain kinds of orders, for example new pickup orders because your fulfillment process prioritizes based on this, using the filters provided can help isolate just those. You can also filter by fulfillment status: New, In Progress, Completed, or Ready for Pick Up. You can also search for a specific order by its Order Number.

Note that expedited orders appear in red in the Order column listing.

When using filters, remember to click the Refresh button to reload the page with just the orders you want to see. Also remember to click Clear Filters when they’re no longer needed, to make sure you see all orders.

 Preferred web browser

While the Order Worklist works well with most browsers, we recommend using Google Chrome. If using Internet Explorer, make sure it’s version 11 or newer.

 About printing labels and shipping documents

Exactly how you print a pick list, shipping label or other shipping documents depends on the browser you’re using, and your organization’s network. There is no specific method for printing these documents from the Order Worklist or other areas of Site Manager. Once you’ve generated and displayed a document you want to print, use your browser’s print feature to make the hardcopies you need.

 About shipping and pickup orders

The processes for fulfilling shipping and pickup orders differ slightly from one another, so they’re presented separately to avoid confusion. There is no actual need to deal with one or the other exclusively. You can certainly fulfill both types in parallel; just be aware which orders fall into either category. Note that pickup orders display the text 'Customer Pickup' in the destination column to avoid confusion between the two.

 About elapsed time and order date

System configurable parameters now support displaying the length of time an order has been active in the Order Worklist, rather than the date it was placed. For the most part, procedures presented here assume that elapsed time is the mode being used.
