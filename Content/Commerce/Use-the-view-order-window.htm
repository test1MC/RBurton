Using the view order window

Click an order number and you're taken to the view order window. From the tabs in this window you can view, print, edit, credit the order, and email the customer. You can also view the item level status and the order history. In addition, you can process payments, view the PO information, re-order, manage shipping, and process an RMAreturn materials authorization.

To open an order in the view order window

  1. In Commerce > Orders > Managing Existing.
  2. Search for the order you want to view. For more about order search options, see Managing existing orders.

  3. Click on an order's Order Number to be taken to the view order window.

The view order window displays a row seven tabs arranged across its top, with the View Order tab active.

Contents and actions available to you in these tabs are:

To return to the Commerce > Orders > Manage page at any time, click the Return button at the top of the window.

When a customer wants to repeat a previous order, use the Re-Order option to manage it using a limited number of steps.

To Re-Order:

  1. Click the Re-Order button on the View Order tab of the order the customer wants to re-order.

    The Re-Order screen pre-populates with the customer and order details from the previous order. This screen functions the same as in Commerce > Orders > Create New.

  2. Add any additional items to the order use the same process describe rd in Commerce > Orders > Create New.
  3. To finalize the order, go through the Checkout screen and verify all information before submitting the order.
  4. Click Place Order to complete the transaction
 Edit Order tab

Use the Edit Order screen to change various aspects of a previously placed order. Items you can edit on the Edit Order screen include:

  • Sales Person
  • Customer's email address/password
  • Billing and Shipping details
  • Order Status
  • Address

When editing a customer's address (Order Summary area, Edit Address link), remember that the Company field pertains to the customer's company name—if there is one—and not the customer's address. If you change a customer record, the company name will be associated to every one of a customer's orders. This is because company name is related to the customer and not their address. If you remove the company name, it is removed from each order.

A window opens with the shipping rate information. This is visible if the My Company Ships Perishable Goods check box is enabled in Commerce > Shipping > Settings > Shipping Settings.

About credit types

There are two types of credit: simpleUsed to apply credit to an entire order. and detailedUsed to apply credit to a specific line item in an order.. Use the appropriate procedure:

To refund a Simple Credit

  1. Choose the Select Refund Reason from the list, and enter explanatory notes in the field provided below the list.

  2. Select a Transaction ID from the list if applicable.

  3. Select the appropriate Amount to Credit radio button:

    1. Simple Credit to credit an entire order, or

    2. Detailed Credit if you need to credit a specific item from a multi-item order.

      Any Previous Credits if any for the order, and Available Balancethe order total including shipping fees, taxes, and any previous credits are displayed below these buttons.

  4. The Credit amountThe Credit total minus the restocking fee is calculated and displayed. You can manually enter the amount you want to credit toward the order total in this field.

    Note: If you want the amount in this field to default to "0" for simple credits:select the Crediting Orders defaults to $0.00 setting at Users > Website Settings > Payments > Credit Card Processing Settings. For example, you may want to enact a policy that when refunds are created, $0/00 is inserted in this field rather than the available credit amount. This configuration can help if you want to ensure that Customer Service Representatives enter a valid amount, rather than using the default available credit amount based on the order history.

  5. Select whether to send an Email to Customer informing them of the order credit.

  6. If it is your business policy to only issue in-store credit and not place a credit back on a credit card, select Store Credit Only; this creates a gift card for the customer to use on a future purchase.

    If you use this functionality, you will need to enable Gift Card use in the Users > Settings > Site Layout > Checkout Page Settings area.
  7. Select whether to record an Offline Credit.

  8. Click Credit to submit the credit request.

    If you do not have the permission level needed to credit an order, the button is labeled Save Proposal. When clicked, it creates a task that appears on the Content > Tasks > Internal Task List page notifying the default user or admin of the task. The default user is someone with 'Order Admin' permissions, who can then approve or reject credit proposals created by CSRs. Approvals will create an order credit.

 Process Payment tab

Use the Process Payment screen to perform various card actions for an order. You can use this screen if an order was edited and the payment amount has changed for more than the original authorized amount. Also use it if an card was declined and the customer needs to provide a new credit card number, or when you need to recalculate an order because a customer was added to or removed from a tier.

To process a payment

  1. To process a payment when viewing an order, click the Process Payment tab at the top of the View Order window.

  2. Choose an option from the Processing Options list.

    Options:
    • Auth/Void 50 Cents – Performs an authorization and void for fifty cents. This tests the validity of a card.
    • Auth/Void Full Amount – Performs an authorization and void for the full amount. This tests the validity and funds availability of the card.
    • Authorize Full Amount – Performs an authorization for the full amount. This places an authorization hold for the funds on the customer's credit card.
    • Capture Selected Events – Perform an authorization and capture funds only for the selected items you've selected from the list of items included in the order.
      • Use New Authorization – Occasionally when processing a payment, the existing authorization may expire. This functionality ignores the existing authorization, and obtains a new authorization and immediately captures the funds.
    • Void Existing Authorization – Voids an existing authorization.

  3. Select All Items by clicking the check box at the top of the list of items in the order, or you can select line items individually if you do not want to process payment for all items in the order at this time.

  4. Click Save.

To manually authorize a payment

  1. If you want to manually authorize one-time payments, click the Enter Manual Authorization link, and a new window opens.

  2. Select each line item that you want to process with its checkbox.

  3. Click Save.

Re-shipping an order

You can easily re-ship an order to a customer without canceling the previous order, and creating a one to replace it. Do this by selecting the Re-Ship link displayed under the tracking number. This feature allows for cases where the customer did not receive the shipment or the item was damaged.

When you click the re-ship link next to a line item on the ship order tab, instead of adding a line item to the current order, a new order and order notes are created on the existing order and on the new order. The notes reference each of the orders.

The new re-shipped order will not contain any of the notes from the original order—including gift card orders—but will contain a reference note that associates it to the original order.

  • The order note added to an old order is: 'Reship order has been created for item: [partno]. The order number is [external order ID of reship order].'
  • The order note added to new re-shipped order is: 'This is a Reship order. Original order number is [external order ID of original order].' This new order will not contain any of the original order notes, including gift card orders, and only contains the reference note linking it to the original.

Re-Shipped orders contain the same customer, shipping and billing information as the original order. The new order has no charges associated with it. The item is priced at $0 with no shipping charges. The new order will be placed in an 'ordered' status, and the item(s) will also be in an 'ordered' status.
