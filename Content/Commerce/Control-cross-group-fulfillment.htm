Controlling cross group fulfillment

Use this page to control your company's cross-group fulfillment by assigning supplier groups to affiliate groups. Once configured, when an order comes into the system with an affiliate, checks this table to see which supplier groups are eligible to fulfill this order. Supplier groups are usually set up according to geographic regions and/or brands. They are managed at Commerce > Suppliers > Add/Edit.

Note: The affiliate groups displayed in the drop-down menu are configured at Marketing > Affiliate > Groups.

Supplier groups are managed at Commerce > Suppliers > Add/Edit.

To control cross-group fulfillment

This page maps to 's cross-group fulfillment table.

  1. In Commerce, select Suppliers > Fulfillment Group Linkage.
  2. Choose a group from the Select Affiliate Group list.
  3. Use the check boxes to assign suppliers to the affiliate. Note that:
    1. Only affiliate groups containing active affiliates appear in the drop-down menu.
    2. Only supplier groups containing active suppliers appear with each individual supplier selected.
    3. By default all supplier groups are unchecked
    4. Supplier groups can be assigned to many different affiliate groups

  4. Click Save before selecting another group from the list to set up.

    You'll see the success message "<Affiliate Group Name> successfully updated."
