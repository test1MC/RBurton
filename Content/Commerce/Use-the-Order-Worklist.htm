Using the Order Worklist

Stores, suppliers/warehouses use this screen to view all open orders assigned to them for fulfillment. It shows items assigned to a supplier and in one of the configured order statuses on the Manage Status TypesOrderStatus.aspx screen so that store associates can view and pick orders as quickly as possible.

In this screen, stores and warehouses can view all open orders assigned to them for fulfillment. The screen shows orders/items matching statuses configured in Commerce > Settings > Manage Status Types, then evaluates Pickup, Shipping, and Transfer filters. All orders matching the statuses selected in the filter are displayed at the shipping address level on the screen.

Suppliers can access this screen from Supplier > Supplier Shipping > Order Worklist. Because the functionality is based upon supplier permissions, the supplier view is limited when viewing the details of an order.

(for Suppliers): When viewing this screen, only those orders containing items assigned to the logged in supplier are shown. If the supplier prints a pick-list from this screen, it will only contain items in the selected orders assigned to the logged in supplier. Suppliers can see only the View Order tab, the Update Item Status tab, and the Ship Order tab, and these will contain only items assigned to the logged in supplier. Suppliers are not able to view the Edit Order, Process Payment, Credit Order, or Send PO tabs.

Information provided at the top of the screen includes:

  • Total Unshipped Orders—a count of all orders assigned to the supplier in statuses matching the statuses selected in the filters.
  • Total Orders Today—all orders assigned to a supplier that were created today and in statuses matching the statuses selected in the filters.
The Exp. column denotes expedited orders.

To search orders
  1. In Commerce, select Orders > Order Worklist.
  2. Use the Status list to filter on the order status you want to see, or select All Statuses.
  3. Choose Item Type filters as needed:
    1. Pickup—displays items being picked-up in store
    2. Shipping—displays all other shipping methods, except for local pickup.
    3. Transfer—displays items transferred from one location to another.

    4. Selecting all three—the default selection—displays pickup, shipping, and transfer orders.

  4. Select whether to show Expedited Only orders; if selected, only orders flagged as expedited are displayed.
  5. Click Refresh, and items matching your filter selections are displayed with the most recent information available.
 To manage orders
