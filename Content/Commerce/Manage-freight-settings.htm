Managing Freight settings

Use this area to define custom freight at the product level. In order to define the custom freight values, you must update your custom freight settings. You can add or edit freight categories here, as well as freight tiers.

To add a custom freight category
  1. In Commerce, select Shipping > Freight.

  2. Enter the name of the custom freight category you wish to create in the blank field at the top of the screen.

  3. Click the plus sign beside this field, and the new category appears in the list of categories.

  4. For each new category, select whether to use Fixed or Tier pricing.

    1. If fixed pricing, enter the fixed amount in the Price/Tier field.
    2. If tier pricing, select the tier from the Price/Tier list.
  5. To override pricing for a category by state:
    1. click the category's Override button.
    2. select a state from the list and click the plus sign; repeat to add as many states as needed.
    3. define prices and tiers as needed.

    4. click Save.

  6. Click Save.

 To add a custom tier
  1. In Commerce, select Shipping > Freight.
  2. In the Freight Tiers area of the screen, select Add New Tier from the list.

  3. Enter a name of the tier you wish to create in the Name field.

  4. Click Save.

  5. With the new tier name selected from the list, click the Add First Tier For Selected Tier link.

  6. Add Low Weight and High Weight values, and Shipping Cost for this tier.
  7. Add more weight tiers as needed.
  8. Click Save.
