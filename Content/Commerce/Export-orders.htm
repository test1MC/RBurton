Exporting orders

Use Export to export orders to an Excel spreadsheet for reporting or printing. This area is also a repository for all of your order exports.

Tip: Always do an export to ensure you are using the most recent spreadsheet version. This will also guide you on the proper format for each column and row, as well as which fields are required.

To export orders

  1. In Commerce, select Orders > Export.
  2. Enter a From Date and Thru Daterange in the format MM/DD/YYYY for orders to export.

  3. Click the Export button .

    The Excel file is created and a link to it appears in the Report column.

  4. Click any report in the list to open it.
  5. Click any report's delete icon to delete it.
