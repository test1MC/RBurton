Configuring regional shipping tiers

You configure regions so they can be selected for Fixed Shipping Tiers by state/province or country. You can assign different shipping rates to different shipping regions. This also gives you the ability to define which states/provinces or countries are assigned to each region. Regions are only configurable for Fixed Shipping Tiers and if you use this functionality for a region, all regions must be set up.

Fixed Shipping Tier regions can be defined by state/province or country. Region settings are useful when defining shipping rates to APO/FPO addresses, Alaska, Hawaii, and other countries.

UPS Ground shipping does not return rates for APO/FPO addresses and some territories. Because of this, you should set up USPS for APO/FPO, and assign it to its own region. Make certain you set it to use USPS rates.
If you use this functionality to assign a region to a shipping rate, all regions must then be configured. For example, if you assign separate shipping regions for Alaska and Hawaii, you must then assign a shipping region for the other 48 states.

​After regions added, assign shipping rates in Commerce > Shipping > Settings.

Adding and managing regions

  1. In Commerce, select Shipping > Regions.
  2. To add a new shipping region to the list, click the (+) Add button. A New Region is added to the page.
  3. To configure the new region, or edit an existing one, click its Manage Region icon.

    1. Edit the Region Name.

    1. If you like, use the Region Covers list to display options in just the area you need.
    2. Select the state, province, or country to add to the region.

    3. Click Save, and close the window.

  4. To delete a shipping region, click its Delete icon.
Make certain that when setting up shipping regions to various countries, the countries can receive the shipping type you have configured. For example, if you set up a country with an economy shipping type, there will be an error if the shipper does not allow economy shipping to that country. See the FedEx International Express Countries Served via US export and US import. For UPS, do not use UPS Standard for regions. Refer to this UPS Standard Shipping article for more information.
