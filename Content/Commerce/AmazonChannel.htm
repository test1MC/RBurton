Configuring the Amazon channel

Use this page to see Amazon channel transaction data to help configure marketing settings for it.

To see Amazon channel transaction data

  1. From Commerce > Channels > Amazon, use the Transmission History to view an archive of transactions through Amazon.com for the last 10 days. The information will be deleted once it is 10 days old or older so this will need to be examined regularly to see if any errors have occurred that need to be addressed. For more in depth information on how to setup your Amazon Integration, check the Training Video Archive under the Support tab of the Site Manager to see a video dedicated specifically to Amazon Integration.
  2. Select a Date From and Date Thru range, keeping in mind that there is no information stored prior to the last 10 days.
  3. Select from the drop-down menu what information you would like the report to include (Order Acknowledged, File Feed Sent, and so on), or select All Amazon Messages for a comprehensive report.
  4. Click Search. If you selected All Amazon Messages, a complete archive of all transactions will appear, whether it’s your orders coming in or your product feeds going out. The information listed includes:
    1. Document number
    2. Date
    3. Type (file)
    4. Status
    5. Messages (Number of)
    6. Successes (Number of)
    7. Errors (Number of)

    8. Warnings

  5. If you see an Error, you can click the magnifying glass to view the specific reasons for that error.

  6. If you’re a current Amazon Seller Central customer, and you’re just beginning to use the Aptos platform, you have the option of Importing the Excel spreadsheets that you have been loading into Amazon on a daily basis into this area of the Site Manager. This will automatically create all of your products as well as map all of your classifications and attributes according to the Amazon Item Classification guidelines. The Excel file Import can be found at the bottom of the screen where it says ICG Import.

Amazon ICG Import Sample can be found on the Support tab. Also, the Amazon sub-category must exist prior to importing. Importing these spreadsheets will automatically map to the existing sub-category.
