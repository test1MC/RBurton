The Saved Orders screen

If Save Proposal was selected when creating an order in Commerce > Orders > Create New, it will be saved and viewable here in Commerce > Orders > Saved Orders.

To manage saved orders

  1. In Commerce > Orders > Saved Orders, use the search functionality to locate the order. You can use any of the fields listed or add a search condition to find a saved order.
  2. When you've found the order you're searching for, select it using the radio button.
  3. Click the > (Retail Order) button. This takes you to Commerce > Orders > Create New screen. You can make edits if necessary and proceed to process the order by clicking the in Commerce > Orders > Saved Orders.button at the bottom of the screen.
  4. If the order is no longer applicable, you can be delete it by selecting it and choosing the delete button at the bottom of the screen.

All orders can also be exported to an Excel file if needed.
